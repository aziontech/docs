## Unlink

The `azion unlink` command disconnects a local repo or project folder from Azion. The use of `--yes` option allows the action to complete without asking for confirmation. To get more information about this command, you can use the `-h` or `--help` option.

## Usage

```bash
azion unlink
```

## Local Options

### help  
The `--help` option provides more information about the `azion unlink` action.

### yes  
The `--yes` option completes the unlink action without asking for confirmation.
