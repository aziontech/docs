## How it works

The *addEventListener* function defines the trigger for the execution of JavaScript code to receive the request data. The event listener type is `fetch` and gets the [fetchEvent](/en/documentation/products/edge-application/edge-functions/runtime-apis/javascript/fetch-event/)

## Syntax

```js
  addEventListener(type, listener)
```

## Properties

- type: `string` - the supported type is `fetch`.

- listener: `function` - the function that handles incoming Azion Cells events.

### Example

```js
  addEventListener("fetch", event => {
    return event.respondWith(
      new Response("Hello world")
    )
  })
```
