## July 8, 2024

### Real-Time Metrics

#### Real-Time Metrics REST APIs

<Tag severity="info" client:only="vue">Deprecated</Tag>

Access to the Real-Time Metrics REST APIs has been discontinued and deprecated. As a result, the `api.azionapi.net/analytics` and `api.azionapi.net/metrics` endpoints are no longer available. To perform data queries and analyze your application's activity, you can access:

- **Real-Time Metrics GraphQL API**, which provides aggregated metrics in real time.
- **Real-Time Events GraphQL API**, which provides raw logs in real time.

Read more about [GraphQL API](/en/documentation/devtools/graphql-api/overview/) and the [First Steps](/en/documentation/devtools/graphql-api/first-steps/) to start using it.

## December 22, 2023

### Marketplace

#### Fingerprint

The new **Fingerprint** integration enables you to capture device and network data to better identify devices accessing your applications.

This integration includes two features:

- A browser-level script (optional) to capture the browser data and set it as a session cookie.
- An edge function (mandatory) to capture connection-related data as well as the “frontend fingerprint” (created by the browser script) to create the session fingerprint.

The collected data is used to generate a unique identifier and add it as a request header in other integrations and use cases.

### GraphQL

Azion now provides a [Grafana plugin integration](https://github.com/aziontech/grafana-plugin) available for local installation. With it, you can use Grafana's interface to create dashboards and complement your metrics visualization. Your dashboards can be created using GraphQL queries. You can use it to view and create:

- Top X metrics (such as IP addresses or blocked countries).
- Security metrics.
- Customized alerts.
- Specific status codes.

It queries data from [Real-Time Metrics](/en/documentation/products/observe/real-time-metrics/) and [Real-Time Events](/en/documentation/products/observe/real-time-events/), which use the [GraphQL APIs](/en/documentation/devtools/graphql-api/overview/).

For more details, check the documentation on [How to integrate Azion with Grafana](/en/documentation/products/guides/observe/integrate-grafana/) and [How to customize a dashboard with the Azion plugin on Grafana](/en/documentation/products/guides/azion-plugin-grafana-custom-dash/).
