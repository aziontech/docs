O [Azion Edge Orchestrator](/pt-br/documentacao/produtos/deploy/edge-orchestrator/) permite releases frequentes e confiáveis de aplicações através de orquestração. Esta documentação aborda a arquitetura e os passos para a entrega automatizada de aplicações usando o Azion Edge Orchestrator, garantindo desempenho ótimo e operações simplificadas.

---

## Arquitetura da entrega e operações de aplicações

:::note
Este fluxo pressupõe que um serviço foi criado para salvar uma página HTML contendo uma lista de produtos. Para mais informações, verifique a seção [Implementação](#implementacao).
:::

![Diagrama de arquitetura do Orchestrator](/assets/docs/images/uploads/infrastructure-arch.png)

### Fluxo de dados da entrega e operações de aplicações

1. O usuário acessa o **Azion Console** para gerenciar os **Edge Nodes**.
2. O usuário associa um **Edge Service** aos **Edge Nodes**.
3. Os **Edge Nodes** são gerenciados e configurados através da **Associação de Edge Service**.
4. O **Orchestrator** orquestra serviços nos **Edge Nodes**.
5. Os **Edge Nodes** agora contêm uma página HTML contendo uma lista de produtos.

## Componentes

- [Azion Console](/pt-br/documentacao/produtos/guias/como-acessar-o-azion-console/): interface para gerenciar e monitorar aplicações e edge services.
- [Azion Edge Orchestrator](/pt-br/documentacao/produtos/deploy/edge-orchestrator/): automatiza o deploy e gerenciamento de aplicações em edge nodes.
- [Edge Nodes](/pt-br/documentacao/produtos/deploy/edge-orchestrator/edge-node/): recursos de computação distribuídos que processam dados mais próximos à fonte.
- [Edge Services](/pt-br/documentacao/produtos/deploy/edge-orchestrator/edge-services/): permite a criação e customização de serviços para orquestração em edge nodes.

## Implementação

1. [Instale o Edge Orchestrator Agent](/pt-br/documentacao/produtos/guias/deploy/instalar-orchestrator-agent/) em seu dispositivo para transformá-lo em um edge node.
2. Autorize o edge node para que possa ser orquestrado.
    - No Console, você pode ver e [autorizar edge nodes](/pt-br/documentacao/produtos/guias/deploy/autorizar-edge-node/).
    - Use a [API da Azion](https://api.azion.com/#e6a19e66-71e9-4b10-b50c-c0e0684bf458) para autorizar um node.
3. [Crie serviços](/pt-br/documentacao/produtos/guias/deploy/criar-edge-service/) que irão salvar uma página HTML nos nodes.
4. Atribua serviços a edge nodes.
    - No Console, [vincule serviços](/pt-br/documentacao/produtos/guias/deploy/vincular-servico/) a seus edge nodes.
    - Use a [API da Azion](https://api.azion.com/#230b2947-a706-4143-ae96-3115b8ecd333) para atribuir um serviço a um edge node.
5. [Acompanhe logs](/pt-br/documentacao/produtos/guias/deploy/verificar-logs/) de seus edge nodes e serviços.
