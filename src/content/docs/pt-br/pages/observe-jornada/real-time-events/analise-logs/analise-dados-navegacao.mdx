Se você usa o [Edge Pulse](/pt-br/documentacao/produtos/edge-pulse/) para coletar dados de navegação adicionando uma das **tags JavaScript do Edge Pulse** disponíveis em suas páginas, você pode analisar os dados coletados no **Real-Time Events**.

<Button href="/pt-br/documentacao/produtos/edge-pulse/#edge-pulse-javascript-tag" text="saiba mais sobre tags JavaScript" variant="secondary"></Button>

---

## Consulte no Real-Time Events

[Real-Time Events](xxx) fornece a fonte de dados do **Edge Pulse**. Ao consultá-lo, você pode realizar uma análise detalhada dos logs brutos coletados pelas tags adicionadas.

As categorias para os dados coletados são:

- Navegação
- Performance
- Experiência de usuário
- Informacionais
- Rede

Com a análise, você pode aprimorar seu conteúdo e experiência do usuário.
