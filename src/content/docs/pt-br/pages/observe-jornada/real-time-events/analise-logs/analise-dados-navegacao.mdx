A Azion possui duas interfaces de usuário: Real-Time Manager e Console, que está em *Preview*. Atualmente, o Console está disponível apenas para planos de Desenvolvedor e novas contas. Este guia detalha passos para a interface de usuário do Real-Time Manager.

Se você usa o [Edge Pulse](/pt-br/documentacao/produtos/observe/edge-pulse/primeiros-passos/) para coletar dados de navegação adicionando uma das **tags JavaScript do Edge Pulse** disponíveis em suas páginas, você pode analisar os dados coletados no **Real-Time Events**.

## Consulte no Real-Time Events

[Real-Time Events](/pt-br/documentacao/produtos/observe/analisar-logs/) fornece a fonte de dados do **Edge Pulse**. Ao consultá-lo, você pode realizar uma análise detalhada dos logs brutos coletados pelas tags adicionadas.

As categorias para os dados coletados são:

- Navegação
- Performance
- Experiência de usuário
- Informacionais
- Rede

Com a análise, você pode aprimorar seu conteúdo e experiência do usuário.
