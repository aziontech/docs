## Consulte no Real-Time Events

[Real-Time Events](/pt-br/documentacao/produtos/observe/analisar-logs/) fornece a fonte de dados do **Edge Pulse**. Ao consultá-lo, você pode realizar uma análise detalhada dos logs brutos coletados pelas tags adicionadas.

As categorias para os dados coletados são:

- Navegação
- Performance
- Experiência de usuário
- Informacionais
- Rede

Com a análise, você pode aprimorar seu conteúdo e experiência do usuário.
