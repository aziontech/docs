Depois de [criar um streaming](/pt-br/documentacao/produtos/guias/como-usar-data-stream/), você também pode usar outros produtos [Observe](/pt-br/documentacao/produtos/observe/) para monitorar como o Data Stream está processando e enviando requisições para o conector configurado.

- [Real-Time Metrics](/pt-br/documentacao/produtos/real-time-metrics/): acesso a gráficos com dados agregados para explorar comportamentos durante períodos mais longos de armazenamento de dados.
- [GraphQL API](/pt-br/documentacao/produtos/graphql-api/visao-geral/): acesso a dados agregados e brutos via API com consultas personalizadas, apenas com os dados solicitados.
- [Real-Time Events](/pt-br/documentacao/produtos/real-time-events/): acesso a logs com dados brutos para realizar análises mais detalhadas.
