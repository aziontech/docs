**Real-Time Events** permite que você consulte e analise logs brutos em tempo real. Cada fonte de dados exibe informações detalhadas para logs de eventos relacionados ao acesso, comportamento e desempenho de produtos conectados às suas edge applications.

Assim que suas aplicações começarem a receber tráfego, você poderá diagnosticar dados por meio de Real-Time Events.

<LinkButton link="/pt-br/documentacao/produtos/observe/real-time-events/" label="saiba mais sobre real-time events" severity="secondary" target="_blank" />

Para começar a gerar dados e diagnosticar seus logs:

- [Crie uma edge application](/pt-br/documentacao/produtos/guias/build/criar-uma-aplicacao/): construa uma edge application e, opcionalmente, configure domínios e módulos, além de edge functions.
- [Proteja uma edge application com WAF](/pt-br/documentacao/produtos/secure/proteja-aplicacao/): use o Edge Firewall e o módulo Web Application Firewall (WAF) para proteger suas aplicações contra ameaças.
- [Configure um stream](/pt-br/documentacao/produtos/guias/como-usar-data-stream/): crie e configure um stream e [escolha variáveis](/pt-br/documentacao/produtos/guias/observe/data-stream-selecionar-variaveis/) para enviar seus dados para outras plataformas.
- [Colete dados de navegação](/pt-br/documentacao/produtos/guias/observe/analisar-dados-navegacao/): use o Edge Pulse para coletar dados de navegação por meio das páginas do seu navegador.
- [Entenda logs](/pt-br/documentacao/produtos/guias/observe/entender-logs/): após criar sua edge application e configurar outros produtos, entenda como analisar os logs exibidos em Real-Time Events.
- [Adicione filtros](/pt-br/documentacao/produtos/guias/observe/adicionar-filtros-events/): defina filtros para consultar resultados mais precisos na inspeção de logs em Real-Time Events.
