**Data Stream** permite que você transmita seus logs de edge applications da Azion para plataformas de processamento de stream, SIEM e big data. Todos os logs são baseados em eventos relacionados a produtos conectados às suas aplicações.

Através de conectores, você pode configurar um streaming para transmitir automaticamente seus logs. Isso permitirá que você tenha um conjunto organizado de logs na plataforma de análise de sua escolha e analise o desempenho do seu conteúdo e aplicações.

<Button href="/pt-br/documentacao/produtos/observe/data-stream/" text="go to Data Stream reference" variant="secondary" target="_blank"></Button>

Para começar a gerar logs e configurá-los em um streaming:

- [Crie uma edge application](/pt-br/documentacao/produtos/guias/build/criar-uma-aplicacao/): crie uma edge application e, opcionalmente, configure domínios e módulos, além de edge functions.
- [Proteja uma edge application com WAF](/pt-br/documentacao/produtos/secure/proteja-aplicacao/): use o Edge Firewall e o módulo Web Application Firewall (WAF) para proteger suas aplicações contra ameaças.
- [Configure um streaming](/en/documentation/products/guides/use-data-stream/): crie e configure um streaming e [escolha variáveis](/en/documentation/products/guides/observe/data-stream-select-variables/) para transmitir seus dados para outras plataformas.
