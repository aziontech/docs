Depois de [criar um streaming](/en/documentation/products/guides/use-data-streaming/), você também pode usar outros produtos [Observe](/pt-br/documentacao/produtos/observe/) para monitorar como o Data Streaming está processando e enviando requisições para o conector configurado.

- [Real-Time Metrics](/pt-br/documentacao/produtos/real-time-metrics/): acesse os gráficos com dados agregados para explorar comportamentos ao longo de períodos mais longos de armazenamento de dados.
- [GraphQL API](/pt-br/documentacao/produtos/graphql-api/visao-geral/): acesse os dados agregados e brutos via API com consultas personalizadas com apenas os dados que você solicita.
- [Real-Time Events](/pt-br/documentacao/produtos/real-time-events/): acesse os logs com dados brutos para realizar análises extensivas.
