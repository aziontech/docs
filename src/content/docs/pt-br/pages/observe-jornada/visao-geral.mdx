Os produtos de Observe da Azion auxiliam você no monitoramento e acesso a informações relacionadas aos seus dados. Você pode analisá-los na Plataforma de Edge Computing da Azion ou conectar seus dados a plataformas de terceiros, melhorando sua compreensão do seu conteúdo, como seus usuários estão acessando elas, obtendo insights sobre desempenho e segurança e tomando decisões mais informadas.

## Real-Time Metrics

Monitore aplicações em tempo real visualizando gráficos e filtrando suas consultas.

Você também pode usar a [API GraphQL](/pt-br/documentacao/produtos/graphql-api/visao-geral/) para consultar seus dados via API e personalizar suas consultas.

![Visão geral do fluxo de uma consulta no Real-Time Metrics](/assets/docs/images/uploads/real-time-metrics-overview.png)

<Button href="/pt-br/documentacao/produtos/observe/monitorar-aplicacoes/" text="ir para monitorar aplicações" variant="secondary" target="_blank"></Button>

## Real-Time Events

Analise logs em tempo real por meio de listas detalhadas de variáveis para entender e diagnosticar comportamentos.

Você também pode usar a [API GraphQL](/pt-br/documentacao/produtos/graphql-api/visao-geral/) para consultar seus dados via API, personalizar suas consultas e criar consultas top X.

![Visão geral do fluxo de uma consulta no Real-Time Events](/assets//docs/images/uploads/real-time-events-overview.png)

<Button href="/pt-br/documentacao/produtos/observe/analisar-logs/" text="ir para analisar logs" variant="secondary" target="_blank"></Button>

## Data Stream

Transmita seus dados em tempo real para plataformas de terceiros para ajudar a entender e analisar o desempenho e comportamento de sua edge application e identificar ameaças de segurança.

![Visão geral do fluxo do Data Stream](/assets/docs/images/uploads/data-stream-overview.png)

<Button href="/pt-br/documentacao/produtos/observe/transmitir-dados/" text="ir para transmitir dados" variant="secondary" target="_blank"></Button>

## Edge Pulse

Colete dados de navegação adicionando tags nas páginas do seu navegador para entender como seu conteúdo está sendo acessado e melhorar a experiência do usuário.

<Button href="/pt-br/documentacao/produtos/guias/observe/analisar-dados-navegacao/" text="ir coletar dados de navegação" variant="secondary" target="_blank"></Button>
