**Real-Time Metrics** permite que você monitore suas aplicações em tempo real. As métricas são exibidas por meio de gráficos que fornecem informações relacionadas ao acesso, comportamento e desempenho de produtos conectados às suas aplicações de borda.

Assim que suas aplicações começarem a receber tráfego, você poderá visualizar dados por meio do Real-Time Metrics.

<Button href="/pt-br/documentacao/produtos/observe/real-time-metrics/" label="ir para referência do real-time metrics" type="primary" theme="light" target="_blank" />

Para começar a gerar dados e visualizar suas métricas:

- [Crie uma application](/pt-br/documentacao/produtos/guias/build/criar-uma-aplicacao/): construa uma application e, opcionalmente, configure domínios e módulos, além de functions.
- [Proteja uma application com WAF](/pt-br/documentacao/produtos/secure/proteja-aplicacao/): use o Firewall e o módulo Web Application Firewall (WAF) para proteger suas aplicações contra ameaças.
- [Configure um stream](/pt-br/documentacao/produtos/guias/como-usar-data-stream/): crie e configure um stream e [escolha variáveis](/pt-br/documentacao/produtos/guias/observe/data-stream-selecionar-variaveis/) para enviar seus dados para outras plataformas.
- [Analise métricas](/pt-br/documentacao/produtos/guias/observe/analisar-metricas/): após criar sua application e configurar outros produtos, entenda como analisar as métricas exibidas no Real-Time Metrics.
- [Adicione filtros](/pt-br/documentacao/produtos/guias/observe/adicionar-filtros-metrics/): defina filtros para consultar resultados mais precisos na visualização de dados no Real-Time Metrics.
