**Real-Time Metrics** permite que você monitore suas aplicações em tempo real. As métricas são exibidas por meio de gráficos que fornecem informações relacionadas ao acesso, comportamento e desempenho de produtos conectados às suas aplicações de borda.

Assim que suas aplicações começarem a receber tráfego, você poderá visualizar dados por meio do Real-Time Metrics.

<Button href="/pt/documentation/products/observe/real-time-metrics/" text="ir para referência de streaming de dados" variant="secondary" target="_blank"></Button>

Para começar a gerar dados e visualizar suas métricas:

- [Crie uma edge application](/pt-br/documentacao/produtos/guias/build/criar-uma-aplicacao/): construa uma edge application e, opcionalmente, configure domínios e módulos, além de edge functions.
- [Proteja uma edge application com WAF](/pt-br/documentacao/produtos/secure/proteja-aplicacao/): use o Edge Firewall e o módulo Web Application Firewall (WAF) para proteger suas aplicações contra ameaças.
- [Configure um streaming](/pt-br/documentacao/produtos/guias/como-usar-data-streaming/): crie e configure um streaming e [escolha variáveis](xxx) para enviar seus dados para outras plataformas.
- [Analise métricas](xxx): após criar sua edge application e configurar outros produtos, entenda como analisar as métricas exibidas no Real-Time Metrics.
- [Adicione filtros](xxx): defina filtros para consultar resultados mais precisos na visualização de dados no Real-Time Metrics.
