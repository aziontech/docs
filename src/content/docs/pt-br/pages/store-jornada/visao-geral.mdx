Seja bem-vindo à jornada dos Produtos Store da Azion. Esta documentação apresenta o [SQL Database](/pt-br/documentacao/produtos/store/edge-sql/) e o [Object Storage](/pt-br/documentacao/produtos/store/edge-storage/), ferramentas poderosas para gerenciamento e processamento eficientes de dados.

## Object Storage

O Object Storage da Azion é um modelo de armazenamento de objetos que organiza os dados em "buckets" para uma organização eficiente. Ele suporta a API compatível com S3, classifica as operações de dados em classes e incorpora a replicação de dados em sua arquitetura. Suas principais características incluem a API compatível com S3, buckets públicos, integração com Functions, distribuição global, integração com a rede da Azion e permissões granulares.

### Casos de Uso e Aplicações do Object Storage

O Object Storage é ideal para backup e recuperação, gerenciamento de conteúdo, mídia interativa, hospedagem de sites estáticos, armazenamento de dados não estruturados e sites Jamstack.

## SQL Database

O Azion SQL Database é uma solução SQL de ponta projetada para edge computing, fornecendo consultas de latência ultra baixa para aprimorar aplicações serverless. É totalmente compatível com ACID e usa o dialeto do SQLite, oferecendo um ambiente familiar aos desenvolvedores. Sua arquitetura implementa Main/Replicas para resiliência de dados, garantindo tolerância a falhas por meio da replicação em toda a Global Infrastructure da Azion.

### Casos de uso e aplicações do SQL Database

O SQL Database é adequado para armazenamento de dados de telemetria em tempo real, análise de logs de acesso, gerenciamento eficiente de dados de acesso e suporte a aplicações que exigem recuperação rápida de dados.
