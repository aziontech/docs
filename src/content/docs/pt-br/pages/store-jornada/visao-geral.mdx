Seja bem-vindo à jornada dos Produtos Store da Azion. Esta documentação apresenta o [Edge SQL](/pt-br/documentacao/produtos/store/edge-sql/) e o [Edge Storage](/pt-br/documentacao/produtos/store/edge-storage/), ferramentas poderosas para gerenciamento e processamento eficientes de dados.

## Edge Storage

O Edge Storage da Azion é um modelo de armazenamento de objetos que organiza os dados em "buckets" para uma organização eficiente. Ele suporta a API compatível com S3, classifica as operações de dados em classes e incorpora a replicação de dados em sua arquitetura. Suas principais características incluem a API compatível com S3, buckets públicos, integração com Edge Functions, distribuição global, integração com a rede da Azion e permissões granulares.

### Casos de Uso e Aplicações do Edge Storage

O Edge Storage é ideal para backup e recuperação, gerenciamento de conteúdo, mídia interativa, hospedagem de sites estáticos, armazenamento de dados não estruturados e sites Jamstack.
