Este guia demonstra como usar o **SQL Database** da Azion para excluir um banco de dados existente usando a API da Azion.

## Exclua um banco de dados

Execute a seguinte requisição `DELETE` no seu terminal, substituindo `[TOKEN VALUE]` pelo seu [personal token](/pt-br/documentacao/produtos/guias/personal-tokens/) e `{id_database}` pelo ID do banco de dados que você deseja excluir:

```bash
curl --location --request DELETE 'https://api.azion.com/v4/edge_sql/databases/{id_database}' \
--header 'Authorization: Token [TOKEN VALUE]'
```

Você deve receber uma resposta semelhante a esta:

```bash
 {"state":"pending","data":null}
```

:::note
Esta ação não pode ser desfeita, mas há um tempo de propagação. Após excluir um banco de dados, você não poderá recuperar suas informações.
:::
