Depois de criar seu banco de dados do SQL Database, você pode recuperar os dados armazenados nele.

<LinkButton link="/pt-br/documentacao/produtos/store/sql-database/" label="Consulte a referência do SQL Database" severity="secondary" />

## Liste dados de uma tabela

Execute a seguinte requisição POST no seu terminal, substituindo [TOKEN VALUE] pelo seu [personal token](/pt-br/documentacao/produtos/guias/personal-tokens/) e `{id_database}` pelo ID da tabela que você deseja recuperar:

```bash
curl --location 'https://api.azion.com/v4/edge_sql/databases/{id_database}/query' \
--header 'Authorization: Token [TOKEN VALUE]' \
--header 'Content-Type: application/json' \
--data '{
    "statements": [
        "SELECT * FROM users;"
    ]
}'
```

Você deve receber a seguinte resposta:

```json
{
  "state": "executed",
  "data": [
    {
      "results": {
        "columns": [
          "id",
          "name"
        ],
        "rows": [
          [
            1,
            "item 1"
          ],
          [
            2,
            "item 2"
          ],
          [
            3,
            "item 3"
          ]
        ]
      }
    }
  ]
}
```
