Esta guia mostra como usar o **SQL Database** da Azion para excluir um registro em seu banco de dados usando a API da Azion.

## Exclua um registro

Execute a seguinte requisição `POST` no seu terminal, substituindo `[TOKEN VALUE]` pelo seu [personal token](/pt-br/documentacao/produtos/guias/personal-tokens/) e `{id_database}` pelo ID do banco de dados do qual você deseja excluir o registro:

```bash
curl --location --request POST 'https://api.azion.com/v4/edge_sql/databases/{id_database}/query' \
--header 'Authorization: Token [TOKEN VALUE]' \
--header 'Content-Type: application/json' \
--data-raw '{
"statements": [
"delete from users where name = '\''name'\'' "
]
}'
```

Você deve receber uma resposta indicando o sucesso ou a falha da operação.
