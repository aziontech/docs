Bem-vindo aos produtos Azion Build - seu portal para aproveitar o poder do edge computing em suas aplicações.

**Edge Application** é um produto Azion que permite que você construa aplicações mais próximas dos usuários finais, oferecendo processamento de baixa latência na edge.

Através da arquitetura de proxy reverso, a Edge Application processa requisições de usuários na edge. Ao direcionar requisições para servidores de origem e utilizar capacidades de cache de edge, as requisições seguintes podem acessar dados armazenados em cache no edge, descentralizando o armazenamento e otimizando as taxas de transferência de dados e eficiência. Você também pode transferir o processamento de respostas da origem para a edge. Esta escalabilidade permite que você se adapte ao crescimento da demanda de tráfego sem sobrecarregar sua infraestrutura e reduzir os custos de manutenção.

<Button href="link_edge_foundations" variant="secondary" text="Ir para Edge Foundations"></Button>

A Azion fornece uma gama de ferramentas para configurar sua application na edge:

- [Real-Time Manager (RTM)](https://manager.azion.com): fornece uma interface para configurar sua application e monitorar dados.
- [Azion CLI](/pt-br/documentacao/produtos/build/link_cli): permite que você interaja com a Plataforma Azion diretamente pela linha de comando.
- [Azion API](https://api.azion.com): permite que você integre e automatize processos de implantação, escalonamento e monitoramento de aplicações.
- [Terraform](/pt-br/documentation/produtos/terraform-provider/): integre sua infraestrutura com a Azion usando Terraform.

Quando você constrói uma edge application, pode determinar a origem do seu conteúdo e configurar o domínio que será usado para entregar sua aplicação aos seus usuários finais.
