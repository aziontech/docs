O **SIEM Azion** permite que você expanda a visibilidade dos acessos realizados as suas aplicações. Ao correlacionar dados disponíveis no [Real-Time Metrics](/pt-br/documentacao/produtos/real-time-metrics/) e no [Real-Time Events](/pt-br/documentacao/produtos/real-time-events/), você poderá ampliar as visões de segurança, volume de acessos, saúde, degradação e predição.

## Dashboards

Cada um dos dashboards possui um conjunto de gráficos e tabelas que possibilitam acompanhar as suas aplicações conforme as diferentes necessidades de observabilidade. Veja a seguir os dashboards disponíveis.

### Overview

Apresenta uma visão geral dos principais dados das requisições feitas para suas edge applications.

Os seguintes gráficos estão disponíveis:

- Requests by Category
- Requests by Status - Upstream Status
- Requests (Total x Blocked)
- Requests per Second (Total x Blocked)
- Blocked Requests by Summary
    - Blocked Requests by URI
    - Blocked Requests by Status
    - Blocked Requests by IP
- Top 20 Summary
    - Requests by Remote Address
    - Requests by ASN
    - URIs by Count
    - Requests by Country
    - Requests by Region
    - Requests by User-Agent - Top 10
- Status Breakdown
    - 2XX
    - Top 100 Requests by URI no QS (Status 2xx)
    - 3XX
    - Top 100 Requests by URI no QS (Status 3xx)
    - 4XX
    - Top 100 Requests by URI no QS (Status 4xx)
    - 5XX
    - Top 100 Requests by URI no QS (Status 5xx)
- Response Time
    - Response time average
    - Response time
    - Request count per request_time range
- Log View

Os seguintes gráficos estão disponíveis:

- Top 20 Remote Port

- Last 20 Headers

    - Requests by Status

### Top Requests Connecting

## Alertas

Podem ser configurados alertas para casos específicos que serão recebidos por email. A criação de alertas abrange:

- Números de requisições por método (GET, POST)
- Casos de erro da origem
