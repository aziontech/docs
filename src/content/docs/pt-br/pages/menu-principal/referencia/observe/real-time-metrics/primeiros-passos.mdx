Antes de começar a usar o **Real-Time Metrics**, confira que você tem uma conta no [Azion Console](https://console.azion.com). Você encontra mais informações na [página de documentação](/pt-br/documentacao/produtos/contas/criar-uma-conta/).

O Real-Time Metrics vem como um produto ativado em todas as contas da Azion. Se você quiser conferir se ele está ativo na sua:

1. Acesse o [Azion Console](https://console.azion.com).
2. No canto superior direito, selecione o **Account menu**, representado pelo avatar.
3. Selecione **Billing & Subscriptions**.
4. Selecione a aba **Subscriptions**.
5. Na seção **Edge Analytics**, confira se o switch para **Real-Time Metrics** está habilitado.

Para acessar o Real-Time Metrics:

1. [No Console](https://console.azion.com), no canto superior esquerdo, selecione **Products menu**, representado por três linhas horizontais.
2. Na seção **OBSERVE**, selecione **Real-Time Metrics NEW**.

Você será redirecionado para a página do Real-Time Metrics, que, por padrão, abrirá na aba Applications.

Para um passo a passo de como usar o produto, veja o guia [Como usar o Real-Time Metrics](/pt-br/documentacao/produtos/guias/usar-real-time-metrics/).
