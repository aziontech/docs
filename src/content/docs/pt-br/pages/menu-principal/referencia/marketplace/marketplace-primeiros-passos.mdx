Para acessar o **Marketplace** da Azion, siga esses passos:

1. Crie uma [conta Azion](/pt-br/documentacao/produtos/contas/criar-uma-conta/).
2. Acesse o [Real-Time Manager (RTM)](https://manager.azion.com/).
3. No canto superior esquerdo, selecione as três linhas horizontais para abrir o **Products menu**.
4. Selecione **Marketplace**.

Agora, você pode navegar por todos os recursos disponíveis no Marketplace.

---

## Encontrar um template

Acesse o [Real-Time Manager (RTM)](https://manager.azion.com/) e selecione a opção **Start with a template** na página inicial.

Isso abrirá a página de criação, onde você pode selecionar o template com o qual deseja começar.

## Encontrar uma integração

Acesse o [Real-Time Manager (RTM)](https://manager.azion.com/) e vá para o **Marketplace**. Para encontrar uma integração no Marketplace, você pode:

- Digitar o nome ou apenas uma palavra-chave da solução na **barra de buscas**. Isso retornará qualquer solução que corresponda ao texto.
- Filtrar por categoria usando a seção de filtros, localizada no lado esquerdo da página inicial do Marketplace.
- Navegar pelos cards na tela para encontrar a integração desejada.
