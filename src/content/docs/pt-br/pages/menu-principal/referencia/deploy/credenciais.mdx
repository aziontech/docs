:::note[nota]
Credenciais são usadas somente no Edge Orchestrator. Para gerar tokens para APIs, visite a [documentação da API](https://api.azion.com/) ou a [documentação de Personal Tokens](/pt-br/documentacao/produtos/gestao-de-contas/personal-tokens/).
:::

Você pode gerar a credencial necessária para usar e autenticar os serviços do Edge Orchestrator através do Real-Time Manager (RTM).

As seguintes opções estão disponíveis ao clicar na credencial criada:

- Copiar o token gerado.
- Deletar sua credencial.
- Editar sua credencial. Aqui, você pode atualizar o nome, a descrição e o status de sua credencial (ativo/inativo).

:::note[nota]
O token será gerado após a credencial ser salva.
:::

Você pode copiar o token com o ícone **Copy token**. A seguinte mensagem aparecerá: **Token copied to the clipboard!**

Você pode deletar o token com o ícone **Delete**. Uma mensagem de confirmação aparecerá na tela.
