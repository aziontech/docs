**Integration Services** é a forma mais rápida e segura para migrar seus conteúdos, aplicações web e APIs para a Azion, não importando a complexidade e o volume de recursos que você utiliza. O time irá prover as documentações necessárias para seu sucesso utilizando a plataforma da Azion.

| Plano de Serviço  | Horas        | Horas adicionais disponíveis |
| ----------------- | ------------ | ---------------------------- |
| Business          | 5 horas/ano  | Sim                          |
| Enterprise        | 20 horas/ano | Sim                          |
| Mission Critical  | 60 horas/ano | Sim                          |

Para obter informações mais detalhadas, consulte as opções de [Planos de Serviço](https://www.azion.com/pt-br/precos/#planos).

Um especialista da Azion estará disponível para apoiar o cliente em todo o processo através de:

1. Ativação dos serviços Azion contratados pelo cliente conforme descritos na *Ordem de Serviço*.
2. Levantamento de requisitos técnicos e de negócios junto ao cliente.
3. Treinamento de uso do Real-Time Manager.
4. Configuração dos serviços da Azion para o cliente.
5. Revisão e otimização de configurações dos serviços Azion de acordo com as demandas do cliente durante o lançamento.

O especialista da Azion irá seguir um processo de integração composto pelas seguintes etapas:
