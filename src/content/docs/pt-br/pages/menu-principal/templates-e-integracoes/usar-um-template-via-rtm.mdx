Criar uma edge application é o primeiro passo para começar a usar a **Plataforma de Edge da Azion**. Esta documentação ajudará você a criar uma nova edge application na Azion usando templates.

Os templates são configurações pré-projetadas e reutilizáveis que ajudam a agilizar o processo de criação de edge applications. Nesse sentido, a Azion oferece uma variedade de templates para atender a diferentes casos de uso.

---

## Obter o template

Acesse o [Real-Time Manager (RTM)](https://manager.azion.com/) e selecione a opção **Start with a template** na página inicial.

Isso abrirá a página de criação, onde você pode selecionar o template com o qual deseja começar.

:::note
Alguns templates são integrados a serviços de terceiros e você precisará cumprir certos requisitos, como criar uma conta, gerar tokens ou definir configurações na plataforma de terceiros. Vá para a [página de guias](/pt-br/documentacao/produtos/guias/#azion-templates) e verifique mais detalhes sobre o template específico que você quer usar.
:::

---

## Configurar o template

Quando selecionar o template e abrir a nova página, clique na aba **Settings** e preencha os campos apresentados para configurar sua aplicação.

Por fim, clique no botão **Next**, localizado no canto inferior direito, para iniciar o processo de implantação.

---

## Implantar o template

Durante a implantação, você poderá acompanhar o processo através de uma janela mostrando os logs. Quando estiver concluído, uma nova página será exibida, confirmando que a edge application foi criada com sucesso.

Esta página mostra as informações básicas de sua aplicação, incluindo nome, logs de implantação e o domínio Azion, bem como algumas recomendações sobre opções avançadas para aprimorá-la.

:::note[nota]
O link de sua edge application permite que você veja como ela fica no navegador. No entanto, leva um certo tempo para propagar e configurá-la nas edge locations da Azion. Pode ser necessário aguardar alguns minutos para que a URL seja ativada e para que a página da aplicação seja efetivamente exibida no navegador.
:::

### Principais configurações

Após a implantação, cada template cria uma nova edge application e seu domínio Azion; em alguns casos, ele também pode criar uma nova edge function. Cada template também estabelece algumas configurações-chave predefinidas para otimizar o desempenho de sua aplicação.

Além disso, para garantir o desempenho ideal de certos templates, pode ser necessário ativar alguns produtos da Azion.

Vá para a [página de guias](/pt-br/documentacao/produtos/guias/#azion-templates) e verifique mais detalhes sobre o template específico que você quer usar.

:::tip[dica]
Para saber mais sobre como editar e personalizar as configurações principais de sus aplicação, acesse o guia sobre [configuração](/pt-br/documentacao/produtos/guias/build/definir-configuracoes-principais/) das edge applications. Você também pode [rodar na rede da Azion](/pt-br/documentacao/produtos/rodar-na-rede-da-azion/), adicionando um domínio personalizado e direcionando seu tráfego.
:::
