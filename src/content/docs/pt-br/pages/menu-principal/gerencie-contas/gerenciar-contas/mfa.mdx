:::caution[importante]
A Azion possui duas interfaces de usuário: Real-Time Manager e Console, que está em estágio de *Preview*. Atualmente, o Console está disponível apenas para planos de Developer e novas contas. Siga os passos de acordo com a interface de usuário que estiver utilizando.

[Multi-Factor Authentication (MFA)](/pt-br/documentacao/produtos/gestao-de-contas/multi-factor-authentication/) permite que você reforce a segurança de acessos à sua conta com uma camada adicional de autenticação.

---

## Habilitando MFA

Você pode habilitar a Multi-Factor Authentication (MFA) ao criar um novo usuário, configurá-la para usuários ativos ou para garantir o acesso à sua conta.

<Tabs client:visible>
    <Fragment slot="tab.console">Console</Fragment>
    <Fragment slot="tab.rtm">Real-Time Manager</Fragment>

<Fragment slot="panel.console">
Para habilitar MFA na sua conta:

1. Acesse o [Azion Console](https://console.azion.com).
2. Clique no **Account Menu** no canto superior direito, representado pelo avatar, e selecione **User Preferences**.
3. Ative a chave **MFA**.
4. Clique em **Save** para finalizar o processo.

Para habilitar MFA para usuários existentes:

1. Acesse o [Azion Console](https://console.azion.com).
2. Clique no **Account Menu** no canto superior direito, representado pelo avatar, e selecione **Users Management**.
3. Selecione o usuário desejado para acessar seu perfil.
4. Ative a chave **MFA**.
5. Clique em **Save** para finalizar o processo.
</Fragment>

<Fragment slot="panel.rtm">
Para habilitar MFA na sua conta:

1. Acesse o [Real-Time Manager (RTM)](https://manager.azion.com/).
2. Clique no **Account Menu** no canto superior direito, representado pelo avatar, e selecione **User Preferences**.
3. Ative a chave **MFA**.
4. Clique em **Save** para finalizar o processo.

Para habilitar MFA para usuários existentes:

1. Acesse o [Real-Time Manager (RTM)](https://manager.azion.com/).
2. Clique no **Account Menu** no canto superior direito, representado pelo avatar, e selecione **Users Management**.
3. Selecione o usuário desejado para acessar seu perfil.
4. Ative a chave **MFA**.
5. Clique em **Save** para finalizar o processo.
</Fragment>

## Configurando um aplicativo mobile MFA 

Para instalar um aplicativo autenticador no seu celular, consulte a loja de aplicativos do seu sistema operacional (como a App Store do iOS ou o Google Play do Android). Você pode usar aplicativos como Google Authenticator ou Microsoft Authenticator, por exemplo.

Depois de baixar o aplicativo e fazer login, siga estas etapas para conectar a conta da Azion ao seu aplicativo autenticador escolhido:

1. Localize e clique no botão **+**.
2. Selecione o tipo de conta que você vai adicionar. A câmera será iniciada.
3. Escaneie o código QR que aparece quando você faz login na Azion.
4. Digite o código de seis dígitos e clique em **Verify Code**.

Com MFA, você deve ter acesso ao seu celular para fazer login na Azion e fornecer o código de 6 dígitos.

## Exigindo o uso de MFA para todos os usuários de uma conta

Para exigir MFA para todos os usuários vinculados a uma conta:

<Tabs client:visible>
    <Fragment slot="tab.console">Console</Fragment>
    <Fragment slot="tab.rtm">Real-Time Manager</Fragment>

<Fragment slot="panel.console">
1. Acesse o [Azion Console](https://console.azion.com).
2. Clique no **Account Menu** no canto superior direito, representado pelo avatar, e selecione **Account Settings**.
3. Em **Authentication Settings**, ative a chave **Enforce Multi-Factor Authentication**.
- Quando você ativa este item, MFA será obrigatório para todos os usuários da conta.
4. Clique em **Save** para finalizar o processo.

Para desativar essa configuração, siga as etapas acima, desative a chave **Enforce Multi-Factor Authentication** e clique em **Save**.
</Fragment>

<Fragment slot="panel.rtm">
1. Acesse o [Real-Time Manager (RTM)](https://manager.azion.com/).
2. Clique no **Account Menu** no canto superior direito, representado pelo avatar, e selecione **Account Settings**.
3. Em **Authentication Settings**, ative a chave **Enforce Multi-Factor Authentication**.
- Quando você ativa este item, MFA será obrigatório para todos os usuários da conta.
4. Clique em **Save** para finalizar o processo.

Para desativar essa configuração, siga as etapas acima, desative a chave **Enforce Multi-Factor Authentication** e clique em **Save**.
</Fragment>

</Tabs>

## Fazendo login na Azion usando MFA

Para fazer login na plataforma da Azion com a Multi-Factor Authentication ativada:

<Tabs client:visible>
    <Fragment slot="tab.console">Console</Fragment>
    <Fragment slot="tab.rtm">Real-Time Manager</Fragment>

<Fragment slot="panel.console">
1. Na página de login do Azion Console, insira suas credenciais de login (email e senha) e clique em **Sign In**.
2. Inicie seu aplicativo autenticador.
3. Digite o código de seis dígitos no espaço especificado no Azion Console e clique em **Verify**.
</Fragment>

<Fragment slot="panel.rtm">
1. Na página de login do RTM, insira suas credenciais de login (email e senha) e clique em **Sign In**.
2. Inicie seu aplicativo autenticador.
3. Digite o código de seis dígitos no espaço especificado no RTM e clique em **Verify**.
</Fragment>

</Tabs>

## Resetando MFA 

Em caso de perda ou roubo do seu celular, você não poderá fazer login na Azion a menos que um **Account owner** redefina a função MFA.

:::note
MFA só pode ser redefinido se a aplicação de MFA não for obrigatória para os usuários da conta.
:::

Para isso, você pode pedir a um usuário **Account owner** para desativar sua função MFA. Caso você seja um **Account owner**:

<Tabs client:visible>
    <Fragment slot="tab.console">Console</Fragment>
    <Fragment slot="tab.rtm">Real-Time Manager</Fragment>

<Fragment slot="panel.console">
1. Acesse o [Azion Console](https://console.azion.com).
2. Clique no **Account Menu** no canto superior direito, representado pelo avatar, e selecione **Users Management**.
3. Selecione o usuário desejado para acessar seu perfil.
4. Desative a chave MFA.
5. Clique em **Save** para finalizar o processo.
</Fragment>

<Fragment slot="panel.rtm">
1. Acesse [Real-Time Manager (RTM)](https://manager.azion.com/).
2. Clique no **Account Menu** no canto superior direito, representado pelo avatar, e selecione **Users Management**.
3. Selecione o usuário desejado para acessar seu perfil.
4. Desative a chave MFA.
5. Clique em **Save** para finalizar o processo.
</Fragment>

</Tabs>

Se você for o único **Account owner** na conta, será necessário entrar em contato com a [equipe de suporte](https://tickets.azion.com/en/support/loginen/support/login) para redefinir o MFA.
