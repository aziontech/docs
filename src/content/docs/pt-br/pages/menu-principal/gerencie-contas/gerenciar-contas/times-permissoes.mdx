:::caution[importante]
A Azion possui duas interfaces de usuário: Real-Time Manager e Console, que está em estágio de *Preview*. Atualmente, o Console está disponível apenas para planos de Developer e novas contas. Siga os passos de acordo com a interface de usuário que estiver utilizando.
:::

Com [Teams Permissions](/pt-br/documentacao/produtos/gestao-de-contas/teams-permissions/), você pode criar e gerenciar times, bem como controlar e configurar direitos de acesso para diferentes times em sua organização.

Para acessar as **Teams Permissions**:

<Tabs client:visible>
    <Fragment slot="tab.console">Console</Fragment>
    <Fragment slot="tab.rtm">Real-Time Manager</Fragment>

<Fragment slot="panel.console">
1. Faça login no [Azion Console](https://console.azion.com/).
2. No canto superior direito da página, selecione o menu do avatar. Este é o **Account menu**.
3. Clique em **Teams Permissions**.
</Fragment>

<Fragment slot="panel.rtm">
1. Faça login no [Real-Time Manager (RTM)](https://manager.azion.com/).
2. No canto superior direito da página, selecione o menu do avatar. Este é o **Account menu**.
3. Clique em **Teams Permissions**.
</Fragment>

</Tabs>

:::note
Você deve ser o proprietário da conta (**Account Owner**) ou ter a permissão **Edit Users** para realizar essas configurações.
:::

## Adicione equipes

Você pode criar equipes antes ou depois de adicionar usuários.

Para adicionar uma nova equipe:

<Tabs client:visible>
    <Fragment slot="tab.console">Console</Fragment>
    <Fragment slot="tab.rtm">Real-Time Manager</Fragment>

<Fragment slot="panel.console">
1. Na tela **Team Permissions**, clique em **+ Team**.
2. Digite um nome para a nova equipe no campo **Name**.
3. Na seção **Permissions**, você verá as caixas de seleção **Available Permissions** e **Chosen Permissions**.
4. Clique na permissão que deseja conceder à equipe em **Available Permissions** e depois na **seta** para movê-la para **Chosen Permissions**.

Ao selecionar permissões, você concede acesso às funções definidas para essa permissão apenas aos usuários dessa equipe.

:::note
As permissões são independentes umas das outras. Por exemplo, a permissão Edit Users não permite que uma equipe veja usuários. Nesse caso, você deve ativar See Users também.
:::
</Fragment>

<Fragment slot="panel.rtm">
1. Na tela **Team Permissions**, clique em **Add team**.
2. Digite um nome para a nova equipe no campo **Name for Team**.
3. Na seção **Permissions**, você verá as caixas de seleção **Available Permissions** e **Chosen Permissions**.
4. Clique na permissão que deseja conceder à equipe em **Available Permissions** e depois na **seta** para movê-la para **Chosen Permissions**.

Ao selecionar permissões, você concede acesso às funções definidas para essa permissão apenas aos usuários dessa equipe.

:::note
As permissões são independentes umas das outras. Por exemplo, a permissão **Edit Users** não permite que uma equipe veja usuários. Nesse caso, você deve ativar **View Users** também.
:::
</Fragment>

</Tabs>

## Ative configurações

Na parte inferior da tela, você encontrará as seguintes opções:

<Tabs client:visible>
    <Fragment slot="tab.console">Console</Fragment>
    <Fragment slot="tab.rtm">Real-Time Manager</Fragment>

<Fragment slot="panel.console">
- O switch do **Status** ativa ou desativa permissões de equipe para todos os usuários atribuídos à equipe.
- Com o botão **Cancel**, você retorna à página **Teams Permissions** e também descarta quaisquer edições.
- Clicando no botão **Save**, você pode salvar suas configurações.

Quando você salva suas configurações, você retorna à tela **Teams Permissions**, onde verá a lista de equipes, suas permissões e se estão *Active* ou não.
</Fragment>

<Fragment slot="panel.rtm">
- **Active**: ativa ou desativa permissões de equipe para todos os usuários atribuídos à equipe.
- **Cancel**: com esta opção, você retorna à página **Teams Permissions** e também descarta quaisquer edições.
- **Save**: uma vez que você terminar suas configurações, clique no botão **Save**.

Quando você salva suas configurações, você retorna à tela **Teams Permissions**, onde verá a lista de equipes, suas permissões e se estão *Active* ou não.
</Fragment>

</Tabs>

## Modifique permissões de equipe

Você pode modificar as permissões da equipe sempre que quiser.

<Tabs client:visible>
    <Fragment slot="tab.console">Console</Fragment>
    <Fragment slot="tab.rtm">Real-Time Manager</Fragment>

<Fragment slot="panel.console">
1. Faça login no [Azion Console](https://console.azion.com/) e selecione **Teams Permissions**
2. Na tela **Teams Permissions**, selecione a equipe desejada.
3. Altere as permissões através das caixas de seleção. 

O processo de seleção é o mesmo que para as caixas de seleção descritas em [Adicione equipes](#adicione-equipes).
</Fragment>

<Fragment slot="panel.rtm">
1. Faça login no [Real-Time Manager (RTM)](https://manager.azion.com/) e selecione **Teams Permissions**
2. Na tela **Teams Permissions**, selecione a equipe desejada.
3. Altere as permissões através das caixas de seleção. 

O processo de seleção é o mesmo que para as caixas de seleção descritas em [Adicione equipes](#adicione-equipes).
</Fragment>

</Tabs>

## Delete equipes

Antes de deletar equipes, você deve remover todos os usuários da equipe.

<Tabs client:visible>
    <Fragment slot="tab.console">Console</Fragment>
    <Fragment slot="tab.rtm">Real-Time Manager</Fragment>

<Fragment slot="panel.console">
1. Faça login no [Console](https://console.azion.com/) e selecione **Teams Permissions**.
2. Na tela **Teams Permissions**, clique no ícone de três pontos no canto direito da equipe que deseja excluir.
3. Ao clicar no ícone, selecione **Delete** e uma mensagem aparecerá pedindo para confirmar a exclusão. Para confirmar, digite a palavra requerida na caixa de texto.
4. Clique no botão **Delete** para confirmar o pedido de exclusão. Você então retornará à tela **Teams Permissions**, onde uma mensagem de confirmação é exibida.
</Fragment>

<Fragment slot="panel.rtm">
1. Faça login no [Real-Time Manager (RTM)](https://manager.azion.com/) e selecione **Teams Permissions**.
2. Na tela **Teams Permissions**, ao passar o cursor sobre as equipes, você verá o ícone **trash can** sendo habilitado. Clique na lixeira para deletar uma equipe.
3. Quando você clica na lixeira, uma mensagem aparecerá pedindo para confirmar a exclusão. Para confirmar, digite a palavra requerida na caixa de texto.
4. Clique no botão **Delete** para confirmar o pedido de exclusão. Você então retornará à tela **Teams Permissions**, onde uma mensagem de confirmação é exibida.
</Fragment>

</Tabs>
