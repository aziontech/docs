:::caution[importante]
A Azion possui duas interfaces de usuário: Real-Time Manager e Console, que está em estágio de *Preview*. Atualmente, o Console está disponível apenas para planos de Developer e novas contas. Siga os passos de acordo com a interface de usuário que estiver utilizando.
:::

Você pode gerar credenciais para usar e autenticar serviços. Siga estas etapas:

<Tabs client:visible>
    <Fragment slot="tab.console">Console</Fragment>
    <Fragment slot="tab.rtm">Real-Time Manager</Fragment>

<Fragment slot="panel.console">
1. Acesse o [Azion Console](https://console.azion.com).
2. Acesse a página **Credentials** no **Account menu** no canto superior direito, representado pelo avatar.
3. Clique no botão **+ Credential**.
4. Escolha um nome para sua credencial.
5. Descreva como ou por quem a credencial será usada no campo **Description** em **Settings**.
- Exemplo: `Credential used for Edge Nodes registration`.
6. Clique no botão **Save**.
</Fragment>

<Fragment slot="panel.rtm">
1. Acesse o [Real-Time Manager (RTM)](https://manager.azion.com/).
2. Acesse a página **Credentials** no **Account menu** no canto superior direito, representado pelo avatar.
3. Clique no botão **+ Credential**.
4. Escolha um nome para sua credencial.
5. Descreva como ou por quem a credencial será usada no campo **Description** em **Settings**.
- Exemplo: `Credential used for Edge Nodes registration`.
6. Clique no botão **Save**.
</Fragment>

</Tabs>

O token será gerado após a credencial ser salva.

Ao clicar na credencial criada na página **Credentials**, você pode:

- Copiar o token gerado.
- Excluir sua credencial.
- Editar sua credencial. Você pode atualizar o *name*, *description*, e *status* (ativo/inativo) da sua credencial.
