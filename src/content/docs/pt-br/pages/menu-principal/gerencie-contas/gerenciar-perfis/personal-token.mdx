[Personal Tokens](/pt-br/documentacao/produtos/gestao-de-contas/personal-tokens/) podem ser usadas para autenticar sua identidade como usuário da Plataforma de Edge da Azion e permitir que você use interfaces e ferramentas como a [Azion API](https://api.azion.com) e o [Azion CLI](/pt-br/documentacao/produtos/azion-cli/visao-geral/).

---

## Criando um personal token

### Via Real-Time Manager

1. Faça login no [Real-Time Manager (RTM)](https://manager.azion.com/).
2. No **Account menu** no canto superior direito, representado por três linhas horizontais, selecione **Personal Tokens**.
3. Clique no botão **Add a personal token**.
4. Preencha os campos obrigatórios:

   - **Personal Token name**
   - **Expires in**. Você pode escolher a data de expiração do token entre 1, 7, 15, 30, 90 dias, 1 ano, ou uma data personalizada.
   - **Personal Token description** (opcional)

5. Clique no botão **Create Token**.
6. Clique no ícone de **visibility** no campo **Personal Token** para ver seu token.
7. Clique no ícone de **copy token** para copiar seu token.

:::note
Você só pode ver seu token quando o cria pela primeira vez.
:::

### Via API

1. Execute o seguinte comando cURL no seu terminal, substituindo `[TOKEN VALUE]` por um personal token já existente:

```
curl --location 'https://api.azionapi.net/iam/personal_tokens' \
--header 'Accept: application/json; version=3' \
--header 'Authorization: Token [TOKEN VALUE]' \
--header 'Content-Type: application/json' \
--data '{   
    "name": "One day token",
    "expires_at": "2023-07-26T00:00",
    "description": "example"
}'
```

Você também pode alterar os campos `name`, `expires_at` e `description`.

:::tip
Confira a [documentação da API da Azion](https://api.azion.com/) e a [especificação OpenAPI](https://github.com/aziontech/azionapi-openapi/) para saber mais sobre todos os recursos disponíveis via API.
:::

## Revogando um personal token

Para revogar seu **Personal Token**, proceda da seguinte maneira:

1. Faça login no [RTM](https://manager.azion.com/).
2. No **Account menu** no canto superior direito, representado por três linhas horizontais, selecione **Personal Tokens**.
3. Passe o mouse sobre o item do personal token que você deseja excluir e clique no ícone da **trash can**.
4. Na janela de verificação, digite **delete**.
5. Clique no botão **Delete**.

Você verá uma mensagem confirmando a exclusão do seu token.
