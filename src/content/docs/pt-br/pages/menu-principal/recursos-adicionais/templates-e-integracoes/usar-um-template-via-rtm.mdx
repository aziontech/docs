Criar uma edge application é o primeiro passo para começar a usar a **Plataforma de Edge da Azion**. Nesse sentido, os templates são configurações pré-projetadas e reutilizáveis que ajudam a agilizar o processo de criação de edge applications. A Azion oferece uma variedade de templates para atender a diferentes casos de uso.

Esta documentação ajudará você a criar uma nova edge application na Azion usando templates.

## Obtenha o template

Para obter um template:

1. Acesse o [Azion Console](https://console.azion.com).
2. Clique no botão **+ Create** na página inicial.
3. No modal, selecione a opção **Templates**.
4. Navegue e selecione o card do template com o qual deseja começar.

:::note
Alguns templates são integrados a serviços de terceiros e você precisará cumprir certos requisitos, como criar uma conta, gerar tokens ou definir configurações. Além disso, pode ser necessário ativar alguns produtos da Azion. Vá para a [página de guias](/pt-br/documentacao/produtos/guias/#azion-templates) e verifique mais detalhes sobre os pré-requisitos do template específico que você quer usar. 
:::

---

## Configure o template

Após a seleção do template que você deseja usar, abrirá a página de configuração. Nesta página, você deve fornecer as informações para configurar sua aplicação.

Por fim, clique no botão **Deploy**, localizado no canto inferior direito, para iniciar o processo de implantação.

:::note
Alguns templates têm integração com o GitHub. Nesse caso, você deve conectar a Azion com sua conta no GitHub.

Quando apresentado, clique no botão **Connect with GitHub**. Isso abrirá um pop-up onde você pode confirmar a conexão e definir suas permissões e acesso ao repositório conforme desejado. 
:::

Durante a implantação, você poderá acompanhar o processo através de uma janela mostrando os logs. Quando estiver concluída, a página mostra informações sobre a aplicação, incluindo nome, logs de implantação e o domínio Azion, bem como algumas recomendações sobre opções avançadas para continuar sua jornada.

:::note[nota]
O link de sua edge application permite que você veja como ela fica no navegador. No entanto, leva um certo tempo para propagar e configurá-la nas edge locations da Azion. Pode ser necessário aguardar alguns minutos para que a URL seja ativada e para que a página da aplicação seja efetivamente exibida no navegador.
:::

### Principais configurações

Após a implantação, cada template cria uma nova edge application e seu domínio Azion; em alguns casos, ele também pode criar uma nova edge function. Cada template também estabelece algumas configurações-chave predefinidas para otimizar o desempenho de sua aplicação.

Vá para a [página de guias](/pt-br/documentacao/produtos/guias/#azion-templates) e verifique mais detalhes sobre o template específico que você quer usar.

:::tip[dica]
Para saber mais sobre como editar e personalizar as configurações principais de sua aplicação, acesse o guia sobre [configuração](/pt-br/documentacao/produtos/guias/build/definir-configuracoes-principais/) das edge applications. Você também pode [rodar na rede da Azion](/pt-br/documentacao/produtos/ative-em-producao/), adicionando um domínio personalizado e direcionando seu tráfego.
:::
