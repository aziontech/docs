Esta documentação conta com os primeiros passos para a utilização da **API da Azion**.

## Visão geral da API

A **API da Azion** é uma *API RESTful* baseada em requisições *HTTP* e *HTTPS*, que permite usuários interagirem com os produtos da Azion. A coleção completa dos métodos e endpoints da API está disponível na documentação oficial.

<Button href="https://api.azion.com/" text="Ir para documentação da API da Azion"></Button>

A maioria dos endpoints da API da Azion possuem os principais métodos HTTP (`GET`, `POST`, `PUT`, `PATCH` e `DELETE`) para que o usuário possa criar, ler, atualizar e excluir instâncias de configurações na Azion. Todas as alterações feitas por meio da API são refletidas no [Real-Time Manager (RTM)](https://manager.azion.com).

:::tip[dica]
Acesse a [Introdução à API](https://api.azion.com/#intro) para encontrar informações de referência sobre a API.
:::

## Pré-requisitos

Para fazer requisições com a **API da Azion**, você precisa de:

- Uma conta no [Real-Time Manager (RTM)](/pt-br/documentacao/produtos/contas/criar-uma-conta/).
- Um [personal token](/pt-br/documentacao/produtos/gestao-de-contas/personal-tokens/).

:::tip[dica]
Veja [como criar uma conta no RTM](/pt-br/documentacao/produtos/contas/criar-uma-conta/) e como gerar um [personal token](/pt-br/documentacao/produtos/gestao-de-contas/personal-tokens/) para usar com a API.
:::

## Faça requisições com a API
 
A documentação da API da Azion oferece exemplos de como fazer requisições utilizando o terminal, o comando curl, ou como automatizar essas requisições, implementando seus snippets em softwares.

Para trocar a linguagem do exemplo da requisição, navegue até `Example Request` e mude de curl para outra linguagem de programação desejada.
