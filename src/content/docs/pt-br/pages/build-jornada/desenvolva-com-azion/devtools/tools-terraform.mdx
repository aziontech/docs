**Azion Terraform Provider** é uma ferramenta poderosa que se integra perfeitamente ao Terraform, a principal plataforma de infraestrutura como código (IaC).

O Terraform permite que você gerencie eficientemente sua infraestrutura por meio de código, possibilitando a reutilização, versionamento e compartilhamento de definições de infraestrutura.

O Azion Terraform Provider é um projeto de código aberto registrado no Terraform Registry. Ele utiliza o SDK da Azion (Go) para se comunicar com as APIs da Azion, permitindo que você gerencie sua infraestrutura hospedada na plataforma da Azion usando a simplicidade e flexibilidade do código.
