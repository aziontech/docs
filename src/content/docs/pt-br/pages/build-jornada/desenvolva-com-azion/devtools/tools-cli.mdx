A **Azion CLI** é uma forma amigável de trabalhar com a Plataforma de Edge da Azion, permitindo que você crie e gerencie aplicações por meio de comandos simples. Ela permite que você inicialize, construa e implante projetos em vários frameworks, tais como:

- Next.js
- Vue
- Angular
- Astro
- Hexo
- Vite
- React

Para começar, você precisa [instalar a Azion CLI](/pt-br/documentacao/produtos/azion-cli/visao-geral/#instalar-a-azion-cli).

Dependendo do sistema operacional do seu computador, você pode escolher um método de instalação simples.

Ela também fornece algumas opções para personalizar sua experiência. Por exemplo, você pode verificar qual versão está usando, ver logs detalhados e ajustar várias configurações para atender às suas necessidades.
