O **Model Context Protocol (MCP)** é um padrão aberto desenvolvido pela Anthropic que permite que assistentes de IA se conectem de forma segura a fontes de dados e ferramentas. O servidor MCP da Azion permite que assistentes de código como Claude, Cursor e Windsurf acessem diretamente as capacidades da Plataforma Azion, fornecendo assistência inteligente para seu fluxo de desenvolvimento.

---

## Como funciona

O MCP atua como uma interface padronizada entre assistentes de IA e serviços externos. Quando você configura seu assistente de código com o servidor MCP da Azion:

1. O assistente conecta-se ao servidor MCP usando seu Personal Token
2. O servidor expõe a documentação, ferramentas e capacidades da Azion
3. Você interage com o assistente usando linguagem natural
4. O assistente usa as ferramentas MCP para buscar documentação, gerar configurações e fornecer guias de deploy

---

## Endpoint da API

O servidor MCP da Azion está disponível em:

**URL de Produção**: `https://mcp.azion.com`

---

## Capacidades

O servidor MCP da Azion fornece nove ferramentas especializadas:

| Ferramenta | Propósito |
|------------|-----------|
| `search_azion_docs_and_site` | Buscar documentação e recursos da Plataforma Azion |
| `search_azion_code_samples` | Recuperar bibliotecas de código e exemplos de implementação |
| `search_azion_cli_commands` | Obter comandos e informações de uso do Azion CLI |
| `search_azion_api_v3_commands` | Acessar documentação da API v3 (legado) |
| `search_azion_api_v4_commands` | Acessar documentação da API v4 (atual) |
| `search_azion_terraform` | Buscar documentação do provider Terraform |
| `create_rules_engine` | Gerar configurações do Rules Engine |
| `create_graphql_query` | Construir queries GraphQL para analytics |
| `deploy_azion_static_site` | Obter guias para deploy de sites estáticos |

---

## Requisitos

- **Personal Token Azion**: Crie um no [Console Azion](https://console.azion.com/personal-tokens)
- **Assistente de código compatível com MCP**: Claude, Cursor, Windsurf ou VS Code
- **Node.js 18+** (para algumas configurações usando `mcp-remote`)
