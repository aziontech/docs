## Azion Bundler

[Azion Bundler](https://github.com/aziontech/bundler) é o *framework adapter* da Azion, construído para ser executado no [Azion Runtime](/pt-br/documentacao/runtime/visao-geral/). É um trabalho em evolução, com o objetivo de suportar vários web frameworks. O projeto é de código aberto e incentiva contribuições da comunidade.

A Azion oferece diferentes maneiras de começar a trabalhar com web frameworks:

- [Azion CLI](/pt-br/documentacao/produtos/azion-cli/visao-geral/)
- [Azion Console](/pt-br/documentacao/produtos/comecar-com-um-template/)

## Astro

Astro oferece um método moderno e amigável para criar websites. Ele combina princípios de desempenho consagrados de décadas de desenvolvimento web com a conveniência das tendências de desenvolvimento baseado em componentes. Você pode usar seu framework JavaScript preferido e, por padrão, o Astro garante que apenas o JavaScript essencial seja enviado ao usuário.

Implementação:

## Eleventy


Eleventy é um gerador de site estático simples que permite escrever usando sua linguagem de template favorita. É altamente flexível e não impõe como você deve estruturar seu projeto.
