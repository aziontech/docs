As **Azion Libraries** fornecem um conjunto de ferramentas para interagir com vários serviços Azion, incluindo Produtos (por exemplo, **Purge** e **Storage**) e Utilitários (como **Image Processor** e **Cookies**). Cada biblioteca é configurável e suporta modo de depuração e configuração baseada em variáveis de ambiente.

Essas bibliotecas foram projetadas para serem versáteis e podem ser usadas tanto dentro quanto fora do ambiente do [Azion Runtime](/en/documentation/products/azion-edge-runtime/). Quando usadas fora do Azion Runtime, as bibliotecas interagem com os serviços Azion via APIs REST. No entanto, quando usadas dentro do Azion Runtime, elas aproveitam as capacidades internas do runtime para desempenho e eficiência aprimorados.

---

## Instale as Azion Libraries
Você pode instalar o pacote usando **NPM** ou **Yarn**:

```bash
npm install azion
```

```bash
yarn add azion
```

## Utilize as Azion Libraries

Para mais informações sobre como usar as **Azion Libraries**, consulte a [documentação](https://www.npmjs.com/package/azion).
