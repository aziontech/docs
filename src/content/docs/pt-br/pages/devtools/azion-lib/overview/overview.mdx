As **Azion Libraries** fornecem um conjunto de ferramentas para interagir com vários serviços da Azion, incluindo **Purge**, **Edge Storage** e **Image Processor**. Cada biblioteca inclusa é configurável e suporta o modo de depuração e a configuração baseada em variáveis de ambiente.

Essas bibliotecas são projetadas para serem versáteis e podem ser usadas tanto dentro quanto fora do ambiente de [Azion Runtime](/pt-br/documentacao/runtime/visao-geral/). Quando usadas fora do Azion Runtime, as bibliotecas interagirão com os serviços da Azion via APIs REST. No entanto, quando usadas dentro do Azion Runtime, as bibliotecas aproveitarão as capacidades internas de execução para um desempenho e eficiência aprimorados.

---

## Instale as Azion Libraries

Você pode instalar o pacote usando **NPM** ou **Yarn**:

```bash
npm install azion
```

```bash
yarn add azion
```
