Inicialize uma edge application escolhendo um template inicial. Você pode escolher um template com base em Next, Vue, React, Angular, Astro e outras opções.

:::note
No caso de um ou mais campos obrigatórios não serem informados através das flags específicas, uma mensagem interativa será exibida, solicitando essas informações.
:::

---

## Uso

```sh
  azion init
```

## Flags opcionais

#### name

A opção `--name` pode ser especificada, mas não é obrigatória. Se não for informado, o nome do seu projeto será gerado automaticamente.

#### help

A opção `--help` exibe mais informações sobre o comando `init`.

:::note 
O comando `init` possui uma jornada interativa. Ele perguntará se você deseja inicializar um servidor de desenvolvimento local e implantar o projeto.
:::
