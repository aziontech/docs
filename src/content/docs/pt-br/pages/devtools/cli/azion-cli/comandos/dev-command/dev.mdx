## dev

O comando `dev` adapta o ambiente para o build de sua application e executa o projeto localmente. É usado para projetos já inicializados ou vinculados a uma **Azion application**.

:::tip
Se você deseja executar este comando para um projeto que ainda não foi inicializado como uma Azion application, você pode executar o comando `link` para vincular o projeto à Azion.
:::

<Button href="/pt-br/documentacao/produtos/azion-cli/link-comando/" label="Vincule um projeto em uma Azion edge appliction" type="primary" theme="light" /> 

:::note 
É necessário rodar o comando dentro da pasta raiz do projeto.
:::

### Uso

```sh
azion dev
```

### Saída 

Depois de inicializar o servidor de desenvolvimento local, a aplicação passa pelo processo de `build`.

```sh 
Building your Applications. This process may take a few minutes
Running build step command:
...
```

Em seguida, quando a build estiver completa, o acesso à aplicação é apresentado:

```sh
[Azion Bundler] [Server] › ✔  success   Function running on port http://localhost:3000  
```

### Flags opcionais

#### help

A opção `--help` exibe mais informações sobre o comando `dev`.

#### firewall

A opção `--firewall` indica que a função é destinada para o Firewall.
