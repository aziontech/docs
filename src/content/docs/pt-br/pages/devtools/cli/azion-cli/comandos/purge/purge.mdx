Azion CLI Purge permite que você realize o purge do cache do seu conteúdo na Azion.

---

## Purge

### Uso

```bash
azion purge --cache-key "www.domain.com/@@cookie_name=cookie_value,www.domain.com/test.js"
```

### Flags opcionais

#### cache-key

A opção `--cache-key` fornece uma lista de URLs com *cache keys* que devem ser excluídas do Edge Cache.

#### help

A opção `--help` exibe mais informações sobre o comando de purge.

#### layer

A opção `--layer` especifica a camada onde o purge será executado. Os valores possíveis são `edge_caching` ou `l2_caching`. O valor padrão é `edge_caching`.

#### urls

A opção `--urls` fornece uma lista de URLs que devem ser excluídas do Azion Edge Cache.

#### wildcard

A opção `--wildcard` especifica a URL Wildcard ou Cache Key para os objetos que você deseja excluir. Apenas uma expressão Wildcard pode ser usada por requisição.
