Com o comando `azion clone [recurso]` você pode clonar:

- [Application](#application).

---

## Application

O comando `azion clone application` cria uma cópia de uma Application existente com um novo nome, preservando sua configuração.

#### Uso

```bash
azion clone application --application-id 1 --name "naruno"
```

#### Flags obrigatórias

##### application-id

A flag `--application-id` especifica o identificador único da Application que você deseja clonar.

##### name

A flag `--name` define o nome que será usado pela nova Application clonada.

#### Flags opcionais

##### help

A opção `-h` ou `--help` exibe mais informações sobre o comando `azion clone application`.

---

## Notas

- A operação de clone cria uma nova Application com base em uma existente.
- Você deve ter permissões apropriadas para clonar Applications.
- Use `azion list edge-application` para encontrar o ID da Application que deseja clonar.
