Acompanhe os logs de suas functions e eventos HTTP de suas applications através da Azion CLI.

---

## Logs de functions

O subcomando `azion logs cells` exibe os logs das functions.

### Uso

```bash
azion logs cells [flags]
```

### Exemplos

```bash
azion logs cells
azion logs cells --tail
azion logs cells --function-id 1234 --limit 10
```

### Flags Opcionais

#### function-id

A flag `--function-id` especifica o ID da função cujos logs você deseja ver. Se não informado, os logs de todas as funções serão exibidos.

#### help

A opção `--help` exibe mais informações sobre o comando `azion logs cells`.

#### limit

A opção `--limit` define quantos logs serão exibidos por requisição. Por padrão, são 100.

#### pretty

A opção `--pretty` exibe os logs de uma maneira mais bonita, facilitando a leitura e análise.

#### tail

A opção `--tail` exibe os logs continuamente, semelhante ao comando `tail -f` em sistemas operacionais semelhantes ao Unix. Isso é útil para monitorar logs em tempo real.

:::note 
Este comando retorna os logs dos últimos 5 minutos.
:::
