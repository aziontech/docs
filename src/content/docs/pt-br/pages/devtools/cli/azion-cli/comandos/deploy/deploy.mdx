Implante uma edge application no edge através da Azion CLI.

:::note 
In case one or more required fields aren't informed through the specific flags, an interactive message prompts and asks for the missing information.
:::

---

## Uso 

```bash
  azion deploy
```

## Flags opcionais

#### path 

A opção `--path` é usada quando é necessário informar o caminho do arquivo para a aplicação estática que está sendo implantada.

#### help

A opção `--help` exibe mais informações sobre o comando `deploy`.
