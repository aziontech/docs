Gerencie seus perfis do Azion CLI para alternar entre diferentes contas e configurações facilmente.

## Uso

```bash
azion profiles [flags]
```

## Descrição

O comando `profiles` exibe informações sobre os perfis que você configurou no seu Azion CLI. Os perfis permitem gerenciar múltiplas contas ou ambientes, facilitando a alternância entre diferentes configurações.

## Exemplos

```bash
$ azion profiles
```

## Flags

### help

A opção `-h` ou `--help` exibe mais informações sobre o comando `profiles`.

## Flags globais

As seguintes flags globais estão disponíveis para o comando `profiles`:

| Flag | Tipo | Descrição | Padrão |
|------|------|-------------|---------|
| `-c, --config` | string | Define a pasta de configuração do Azion para o comando atual apenas, sem alterar as configurações persistentes. | - |
| `-d, --debug` | - | Exibe logs no nível de debug. | - |
| `--format` | string | Altera o formato de saída passando o valor json para a flag. | - |
| `-l, --log-level` | string | Define o nível de log: "debug", "info" ou "error". | "info" |
| `--no-color` | - | Desabilita a saída colorida. | - |
| `--out` | string | Exporta a saída para o `<caminho_arquivo/nome_arquivo.ext>` informado. | - |
| `-s, --silent` | - | Silencia completamente os logs; usado principalmente para automação. | - |
| `--timeout` | int | Define quanto tempo em segundos o CLI aguardará antes de expirar a conexão HTTP. | 50 |
| `-t, --token` | string | Salva um Personal Token localmente para autorizar comandos CLI. | - |
| `-y, --yes` | - | Responde automaticamente "sim" para todas as interações de sim/não. | - |
