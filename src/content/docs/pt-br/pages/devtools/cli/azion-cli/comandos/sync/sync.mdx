Sincronize seu arquivo local contendo a configuração dos recursos de sua aplicação existente com os recursos remotos através da Azion CLI.

## Uso

```bash
azion sync
```

## Flags opcionais

#### config-dir

A opção `--config-dir` é usada para especificar o caminho relativo onde seus arquivos personalizados `azion.json` e `args.json` estão armazenados. O valor padrão é "azion".

#### help

A opção `--help` exibe mais informações sobre o comando `sync`.
