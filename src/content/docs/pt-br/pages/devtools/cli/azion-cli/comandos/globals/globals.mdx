Comandos da Azion CLI que podem ser utilizados em qualquer etapa do processo.

---

## version 

```sh 
azion --version
```

A opção `-v` ou `--version` informa qual versão da Azion você tem instalada.

## debug 

```sh 
azion --debug
```

A opção `-d` ou `--debug` exibe logs no nível de depuração.

## config 

```sh 
azion --config
```

A opção `-c` ou `--config` define a pasta de configuração da Azion apenas para o comando atual, sem alterar as configurações persistentes.

## log-level 

```sh 
azion --log-level info
```

A opção `-l` ou `--log-level` informa o nível de log a ser exibido. As opções são:

- `info` (padrão)
- `error`
- `debug`

## silent 

```sh 
azion --silent
```

A opção `-s` ou `--silent` silencia completamente os logs. Normalmente, é usada para fins de automação.

## token 

```sh 
azion --token seu-token-pessoal
```

A opção `-t` ou `--token` salva um personal token fornecido localmente para autorizar comandos do CLI.

## yes 

```sh 
azion --yes
```

A opção `-y` ou `--yes` responde automaticamente com sim a todas as interações sim/não.

## help

A opção `h` ou `--help` exibe mais informações sobre o comando `azion`.
