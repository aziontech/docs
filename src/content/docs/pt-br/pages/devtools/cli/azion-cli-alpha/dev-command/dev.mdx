## dev

O comando `dev` adapta o ambiente para a build de sua edge application e executa o projeto localmente. É usado para projetos já inicializados ou vinculados a uma Azion edge application.

<Button href="/pt-br/documentacao/produtos/azion-cli/link-comando/" text="vincule um projeto em uma Azion edge appliction" variant="secondary"> 
</Button>

:::note 
É necessário rodar o comando dentro da pasta raiz do projeto.
:::

### Uso

```sh
azion dev
```

### Saída 

Depois de inicializar o servidor de desenvolvimento local, a aplicação passa pelo processo de `build`.

```sh 
Building your Edge Application. This process may take a few minutes
Running build step command:
...
```

Em seguida, quando a build estiver completa, o acesso à aplicação é apresentado:

```sh
[Vulcan] [Server] › ✔  success   Function running on port http://localhost:3000  
```

### Flags opcionais

#### help

A opção `--help` exibe mais informações sobre o comando `dev`.
