## link

O processo de vinculação adapta o projeto para sua vinculação a uma edge application.

:::note 
É necessário executar o comando dentro da pasta raíz do seu projeto.
:::

### Uso

```sh
azion link
```

### Saída 

Confirme se deseja vincular o projeto a uma edge application:

```sh 
? Do you want to link /cli-examples/effervescent_moon to Azion? (y/N) 
```

Escolha o *preset* e o *mode*:

```sh 
? Choose a preset and mode:  [Use arrows to move, type to filter]
> Html (Deliver)
  Javascript (Compute)
  Typescript (Compute)
  Angular (Deliver)
  Astro (Deliver)
  Hexo (Deliver)
  Next (Compute)
```

Diferentes processos serão executados dependendo do template escolhido. É necessário responder as interações do processo.

Escolha se você deseja rodar o projeto localmente:

```sh
? Do you want to start a local development server? (y/N) 
```

:::note 
Se você desejar rodar o projeto localmente em outro momento, execute 'azion dev'.
:::

```sh
? Do you want to deploy your project? No
```

:::note 
Se você desejar implantar o projeto localmente em outro momento, execute 'azion deploy'.
:::

### Flags opcionais

#### auto

A opção `--auto` opode ser especificada,mas não é obrigatória. Se for informada, todo fluxo do comando é executado sem interrupções.

#### name

A opção `--name` pode ser especificada, mas não é obrigatória. Se não for informado, o nome do seu projeto será gerado automaticamente.

#### mode

A opção `--mode` pode ser especificada, mas não é obrigatória. Os valores esperados são `deliver` e `compute`.

#### preset

A opção `--preset` pode ser especificada, mas não é obrigatória. Os valores esperados são:

- Html
- Javascript
- Typescript
- Angular
- Astro
- Hexo
- Next 
- React
- Vue
- static

#### help

A opção `--help` exibe mais informações sobre o comando `link`.
