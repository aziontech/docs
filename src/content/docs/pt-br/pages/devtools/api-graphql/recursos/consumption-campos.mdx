Cada [conjunto de dados da GraphQL API](/pt-br/documentacao/devtools/graphql-api/recursos/#conjuntos-de-dados) tem campos específicos que podem ser usados ao consultar esses dados.

Verifique cada campo disponível e suas descrições abaixo.

:::note
Os dados de consumo (Consumption) são retidos e ficam disponíveis por até *24* meses.

Se você estiver conectado com uma conta do tipo group ou reseller, poderá consultar o consumo de todas as contas de clientes que fazem parte da sua.
:::

---

## workloadConsumptionMetrics (Consumption)

| Campo        | Descrição |
|--------------|-----------|
| accounted    | Quantidade total de uso do produto registrada para o cliente. A resposta para este campo é o resultado de uma soma. Exemplo: `1767504904`. |
| clientId | Identificador único do cliente na Azion. Exemplo: `8437r`. |
| countryCode | Código do país associado aos dados de uso. Refere-se às edge locations onde o processamento ocorre, e não à localização de acesso do usuário. Exemplo: `BR`. |
| metricName   | Nome da métrica medida para análise. Exemplo: `invocations`. |
| productId | Identificador único do produto em uso. Exemplo: `1441110021`. |
| region | Região geográfica associada aos dados de uso. Refere-se às localizações de edge onde o processamento ocorre, e não à localização de acesso do usuário. Exemplo: `latam`. |
| workloadId | Identificador da carga de trabalho associada ao uso. Exemplo: `4829301746`. |

### Sobre os campos countryCode e region

Os campos `countryCode` e `region` indicam as edge locations onde ocorre o processamento de dados, determinadas pela arquitetura de rede e roteamento. É importante observar que essa localização pode não ser necessariamente a mesma de onde o usuário acessa o serviço ou a origem dos dados.

O dataset `workloadConsumptionMetrics` usa as seguintes regiões e países ao recuperar resultados relacionados aos dados de uso:

| Region | Country |
|---------------|-------------|
| All Other Regions | Angola, South Africa  |
| Brazil | Brazil |
| Canada | Canada |
| Europe | Deutschland, Spain, France, Great Britain, Italy, Netherlands, United Kingdom  |
| Latam  | Argentina, Bolívia, Chile, Colombia, Ecuador, Mexico, Peru, Paraguay, Uruguay, Venezuela |
| No Region | No Region |
| North America  | Canada, United States |
| United States | United States |
