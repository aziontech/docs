## Desinstale o agente

Para desinstalar o Azion Edge Orchestrator Agent:

1. Acesse o dispositivo (edge node) do qual você deseja remover o agente.
2. Abra o terminal e vá para a pasta onde o agente está instalado.
3. Execute:

```bash
sudo ./edge-orchestrator uninstall 
```

Saída: 

```bash 
Edge Orchestrator was uninstalled successfully
```

A partir deste momento, seu edge node não receberá mais atualizações.

:::note 
Desinstalar o agente do seu dispositivo não remove o edge node da lista de nodes disponíveis. Para completar o processo, você precisa [excluir seu edge node](#exclua-um-edge-node) da lista de edge nodes disponíveis.
:::

## Exclua um edge node

:::caution[atenção]
Uma vez que um edge node é deletado, ele não pode ser recuperado.
::: 

### Via Azion Console

1. [Acesse o Azion Console](/pt-br/documentacao/produtos/guias/como-acessar-o-rtm/) > **Edge Nodes**.
2. Encontre o edge node que você deseja remover.
3. Passe o cursor sobre o edge node e clique no ícone de uma lixeira.
4. Confirme a exclusão digitando a palavra `delete`.


### Via API 

1. Execute a seguinte requisição `GET` no seu terminal, substituindo `[TOKEN VALUE]` pelo seu [personal token](/pt-br/documentacao/produtos/guias/personal-tokens/) e recupere o ID do edge node que você deseja deletar:

```bash 
curl --location 'https://api.azionapi.net/edge_nodes/' \
--header 'Accept: application/json; version=3' \
--header 'Authorization: Token [TOKEN VALUE]'
```

2. Execute a seguinte requisição `DELETE` no seu terminal, substituindo `[TOKEN VALUE]` pelo seu [personal token](/pt-br/documentacao/produtos/guias/personal-tokens/) e `:id` com o ID do edge node:

```bash 
curl --location --request DELETE 'https://api.azionapi.net/edge_nodes/:id' \
--header 'Accept: application/json; version=3' \
--header 'Authorization: Token [TOKEN VALUE]'
```

:::tip
Confira a [documentação da API da Azion](https://api.azion.com/) e a [especificação OpenAPI](https://github.com/aziontech/azionapi-openapi/) para saber mais sobre todas as funcionalidades disponíveis via API.
:::
