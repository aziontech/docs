Para iniciar o processo de instalação do edge node, você deve baixar o binário de instalação do **Orchestrator** de sua escolha.

Baixe a opção que corresponde ao seu sistema operacional:

| Sistema Operacional | Arquitetura | Arquivo                                                         |
| :--------------- | :----------: | ------------------------------------------------------------ |
| FreeBSD          |    x86_64    | [edge-orchestrator](https://downloads.azion.com/freebsd/x86_64/edge-orchestrator) |
| FreeBSD          |    ARM64     | [edge-orchestrator](https://downloads.azion.com/freebsd/arm64/edge-orchestrator) |
| Linux            |    x86_32    | [edge-orchestrator](https://downloads.azion.com/linux/x86_32/edge-orchestrator) |
| Linux            |    x86_64    | [edge-orchestrator](https://downloads.azion.com/linux/x86_64/edge-orchestrator) |
| Linux            |    ARM32     | [edge-orchestrator](https://downloads.azion.com/linux/arm32/edge-orchestrator) |
| Linux            |    ARM64     | [edge-orchestrator](https://downloads.azion.com/linux/arm64/edge-orchestrator) |
| MacOS            |    x86_64    | [edge-orchestrator](https://downloads.azion.com/darwin/x86_64/edge-orchestrator) |

Alternativamente, você pode baixá-lo usando a linha de comando. No exemplo, o link de download se refere ao agente para `Linux/x86_64`. Se você quiser baixar outra versão, basta alterar a URL para o sistema operacional ou arquitetura necessários:

`curl -O http://downloads.azion.com/linux/x86_64/edge-orchestrator`

:::note
Os comandos executados no Orchestrator Agent devem ser executados com privilégios de administrador. Pode ser necessário executar os comandos com `sudo`.
:::

Após o download, siga as etapas abaixo para instalar o Orchestrator Agent em seu dispositivo:

1. Dentro do diretório onde o binário está, instale o agente do **Orchestrator**:

```bash
sudo chmod +x edge-orchestrator
````

```bash 
sudo ./edge-orchestrator install
```

2. Insira o personal token quando solicitado.
3. Inicie o Orchestrator Agent após instalá-lo:

```bash
sudo ./edge-orchestrator start
``` 

:::note
Se o seu sistema operacional não tiver um gerenciador de serviços (como `systemd`), você deve executá-lo em primeiro plano.
:::

```bash
sudo ./edge-orchestrator start --foreground
```

:::note
Os logs para executar o Orchestrator Agent são armazenados no diretório de instalação, em `/var/log/azion/edge-orchestrator.log`
:::
