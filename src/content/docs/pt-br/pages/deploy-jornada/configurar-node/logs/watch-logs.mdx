Após a instalação do **Orchestrator Agent**, você pode verificar os [logs do seu edge node](#verifique-os-logs-do-edge-node) e os [logs dos triggers dos edge services](#verifique-os-logs-dos-edge-services).

--- 

## Verifique os logs do edge node

1. Acesse o seu [edge node](/pt-br/documentacao/produtos/guias/deploy/instalar-orchestrator-agent/).
2. Abra o terminal. 
3. Execute o seguinte comando: 

```bash
sudo vim /var/log/azion/edge-orchestrator.log 
```

:::note 
Você pode escolher o editor de texto que desejar, desde que esteja instalado no edge node.
:::

Este comando exibe os logs relacionados ao edge node em específico, exibindo:

- A versão do manifesto. 
- Comunicação com o servidor. 
- Erros.

--- 

## Verifique os logs dos edge services

1. Acesse o seu edge node.
2. Abra o terminal. 
3. Execute o seguinte comando:  

```bash
sudo vim /var/log/azion/edge-services.log 
```

Este comando exibe os logs relacionados ao edge node em específico, exibindo os logs dos triggers de instalação, atualização e exclusão configurados nos edge services vinculados ao edge node.
