Você pode desvincular um edge service de um edge node usando:

- [Azion Console](/pt-br/documentacao/produtos/guias/como-acessar-o-rtm/)
- [Azion API](https://api.azion.com/#32f34add-9e9f-4b25-94c9-3cee4c266a23)

---

## Via Azion Console

1. [Acesse o Azion Console](/pt-br/documentacao/produtos/guias/como-acessar-o-rtm/) > **Edge Nodes**.
2. Selecione o edge node do qual você deseja desvincular um serviço.
3. Vá para a aba **Services**.
4. Passe o cursor sobre o serviço na lista e clique no ícone de lixeira.

---

## Via API 

1. Execute a seguinte requisição `GET` no seu terminal, substituindo `[TOKEN VALUE]` com seu [personal token](/pt-br/documentacao/produtos/guias/personal-tokens/) para recuperar o ID do edge node do qual deseja desvincular o edge service:

```bash 
curl --location 'https://api.azionapi.net/edge_nodes/' \
--header 'Accept: application/json; version=3' \
--header 'Authorization: Token [TOKEN VALUE]'
```

2. Execute a seguinte requisição `LIST` no seu terminal, substituindo `[TOKEN VALUE]` com seu [personal token](/pt-br/documentacao/produtos/guias/personal-tokens/) e `:id` pelo ID do edge node para obter o ID da relação entre o edge node e o serviço escolhido:

```bash
curl --location 'https://api.azionapi.net/edge_nodes/:id/services' \
--header 'Accept: application/json; version=3' \
--header 'Authorization: Token [TOKEN VALUE]'
```

3. Execute a seguinte requisição `DELETE` no seu terminal, substituindo `[TOKEN VALUE]` com seu [personal token](/pt-br/documentacao/produtos/guias/personal-tokens/), `:node_id` pelo ID do edge node e `:bind_id` com o ID da relação entre o edge node e o edge service:

```bash
curl --location --request DELETE 'https://api.azionapi.net/edge_nodes/:node_id/services/:bind_id' \
--header 'Accept: application/json; version=3' \
--header 'Authorization: Token [TOKEN VALUE]'
```

Agora, seu edge node não tem mais uma relação com este edge service específico. 

:::tip
Consulte a [documentação da API da Azion](https://api.azion.com/) e a [especificação OpenAPI](https://github.com/aziontech/azionapi-openapi/) para saber mais sobre todos os recursos disponíveis via API.
:::
