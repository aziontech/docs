Você pode criar um edge service usando:

- [Real-Time Manager (RTM)](/pt-br/documentacao/produtos/guias/como-acessar-o-rtm/)
- [Azion API](https://api.azion.com/)

---

## Via RTM 

Para criar um edge service, proceda da seguinte maneira:

1. Vá para [Real-Time Manager (RTM)](/pt-br/documentacao/produtos/guias/como-acessar-o-rtm/) > **Edge Libraries** > **Edge Services**.
2. Clique no botão **Add Service**.
- Seu serviço será criado automaticamente.
- Você pode alterar o nome do serviço clicando em **My New Service** no campo de nome.
3. Vá para a aba **Resources** e clique em **Add Resource**.
4. Configure os recursos de que você precisa para o seu serviço, usando os gatilhos: **Install**, **Reload** ou **Uninstall**.
5. **Opcional**: onde você usou variáveis no conteúdo de um ou mais recursos, você pode definir os valores padrão para eles na aba **Environment**.
- As variáveis devem estar no formato `Variável = Valor`, onde `Variável` foi usada no conteúdo de resources que já estão registrados.
6. Ative o serviço habilitando o switch no canto inferior esquerdo na aba **Environment**.
7. Clique no botão **Save**.

--- 

## Via API

1. Execute a seguinte requisição `POST` no seu terminal, substituindo `[TOKEN VALUE]` pelo seu [token pessoal](/pt-br/documentacao/produtos/guias/personal-tokens/) e informando o nome do serviço a ser criado:

```bash
curl --location 'https://api.azionapi.net/edge_services/' \
--header 'Accept: application/json; version=3' \
--header 'Authorization: Token [TOKEN VALUE]' \
--header 'Content-Type: application/json' \
--data '{
"name": "string"
}'
```

Agora, o serviço está disponível e pronto para ser implantado nos edge nodes disponíveis.

:::tip
Confira a [documentação da API da Azion](https://api.azion.com/) e a [especificação OpenAPI](https://github.com/aziontech/azionapi-openapi/) para saber mais sobre todos os recursos disponíveis via API.
:::
