:::caution[importante]
A Azion possui duas interfaces de usuário: Real-Time Manager e Console, que está em estágio de *Preview*. Atualmente, o Console está disponível apenas para planos de Developer e novas contas. Siga os passos de acordo com a interface de usuário que estiver utilizando.
:::

Para [instalar o Edge Orchestrator Agent](/pt-br/documentacao/produtos/guias/deploy/instalar-orchestrator-agent/) e começar a orquestrar, antes você deve criar suas credenciais. Você pode fazer isso através de:

- [Azion Console](/pt-br/documentacao/produtos/guias/como-acessar-o-azion-console/)
- [Azion Real-Time Manager (RTM)](https://manager.azion.com/)
- [Azion API](https://api.azion.com/)

<Fragment slot="panel.console">
1. Acesse o [Azion Console](/pt-br/documentacao/produtos/guias/como-acessar-o-azion-console/).
2. No canto superior direito, abra o **Account menu** e selecione **Credentials**.
3. Clique no botão **+ Credential**.
4. Dê um nome fácil de lembrar para sua credencial.
5. Opcionalmente, descreva como ou por quem a credencial será usada no campo **Description**.
  - Exemplo: `Credencial usada para registro de Edge Nodes`.
6. Clique no botão **Save**.

Um token será gerado após a credencial ser salva.

Ao clicar na credencial criada na página **Credentials**, você pode:

- Copiar o token gerado.
- Excluir sua credencial.
- Editar sua credencial. Você pode atualizar o *nome*, a *descrição* e o *status* (ativo/inativo) da sua credencial.
</Fragment>

<Fragment slot="panel.rtm">
1. Acesse o [Real-Time Manager (RTM)](https://manager.azion.com/).
2. No canto superior direito, abra o **Account menu** e selecione **Credentials**.
3. Clique no botão **Add Credential**.
4. Dê um nome fácil de lembrar para sua credencial.
5. Opcionalmente, descreva como ou por quem a credencial será usada no campo **Description**.
  - Exemplo: `Credencial usada para registro de Edge Nodes`.
6. Clique no botão **Save**.

Um token será gerado após a credencial ser salva.

Ao clicar na credencial criada na página **Credentials**, você pode:

- Copiar o token gerado.
- Excluir sua credencial.
- Editar sua credencial. Você pode atualizar o *nome*, a *descrição* e o *status* (ativo/inativo) da sua credencial.
</Fragment>

<Fragment slot="panel.api">
1. Execute a seguinte requisição `POST` no seu terminal: 

```bash 
curl --location 'https://api.azionapi.net/credentials/' \
--header 'Accept: application/json; version=3' \
--header 'Authorization: Token [TOKEN VALUE]' \
--header 'Content-Type: application/json' \
--data '{
  "description": "credential to use with Orchestrator",
  "name": "new credential",
  "status": true
}'
```

Substitua `[TOKEN VALUE]` pelo seu [personal token](/pt-br/documentacao/produtos/guias/personal-tokens/) e informe as seguintes propriedades no corpo da requisição:

| Propriedade | Descrição | Obrigatório | 
| - | - | - | 
| `name` | Nome da credencial que você deseja criar | Sim |
| `description` | Campo aberto para descrever sua credencial com informações sobre seu propósito | Não |
| `status` | Status da sua credencial | Não  |

Agora você pode [instalar o Edge Orchestrator Agent](/pt-br/documentacao/produtos/guias/deploy/instalar-orchestrator-agent/) no seu edge node e começar a orquestrar seus serviços.

:::tip
Confira a [documentação da API da Azion](https://api.azion.com/) e a [especificação OpenAPI](https://github.com/aziontech/azionapi-openapi/) para saber mais sobre todos os recursos disponíveis via API.
:::
