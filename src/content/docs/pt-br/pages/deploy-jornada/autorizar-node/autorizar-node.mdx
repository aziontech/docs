Se você já instalou o [Azion Orchestrator Agent]() no seu dispositivo, o próximo passo é autorizar o edge node na plataforma. Você pode fazer isso usando:

- [Real-Time Manager (RTM)](/en/documentation/products/guides/how-to-access-rtm/)
- [API Azion](https://api.azion.com)

<Button href="/en/documentation/products/guides/deploy/install-orchestrator-agent/" text="instalar o orchestrator agent" variant="secondary"></Button>

---

## Via RTM

1. [Acesse o RTM](/en/documentation/products/guides/access-rtm/).
2. No canto superior esquerdo da página, abra o **Products menu**, representado por três linhas horizontais, e selecione **Edge Nodes**.
3. Encontre o edge node que você deseja autorizar, que terá o status `Aguardando Autorização`.
4. Passe o mouse sobre o edge node e clique no ícone de uma chave.
5. Confirme a autorização.

---

## Via API 

1. Execute a seguinte requisição `GET` no seu terminal, substituindo `[TOKEN VALUE] pelo seu [token pessoal](/en/documentation/products/guides/personal-tokens/):

```bash 
curl --location 'https://api.azionapi.net/edge_nodes/' \
--header 'Accept: application/json; version=3' \
--header 'Authorization: Token [TOKEN VALUE]'
```


2. Execute a seguinte requisição `PATCH` no seu terminal, substituindo `[TOKEN VALUE]` pelo seu [token pessoal](/en/documentation/products/guides/personal-tokens/) e informando o ID do edge node:

```bash 
curl --location --request PATCH 'https://api.azionapi.net/edge_nodes/authorize' \
--header 'Accept: application/json; version=3' \
--header 'Authorization: Token [TOKEN VALUE]' \
--header 'Content-Type: application/json' \
--header 'accept: application/json; version=3' \
--data '{
    "edge_node_ids": [
        0
    ]
}'
```


Agora, seu edge node está autorizado e capaz de trabalhar com [edge services](/en/documentation/products/deploy/edge-orchestrator/edge-services/). 

:::tip
Confira a [documentação da API Azion](https://api.azion.com/) e a [especificação OpenAPI](https://github.com/aziontech/azionapi-openapi/) para saber mais sobre todos os recursos disponíveis via API.
:::
