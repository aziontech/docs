**Edge Orchestrator** permite que você gerencie e controle recursos de edge em tempo real, incluindo a implantação, atualização e gerenciamento de vários serviços e recursos.

A Azion oferece duas maneiras de gerenciar edge nodes, edge services, recursos e credenciais:

- [Azion API](https://api.azion.com/#9a7daab9-63f7-43a9-9959-2bd91088fca8)
- [Azion Console](/pt-br/documentacao/produtos/guias/como-acessar-o-rtm/)

:::note 
A criação de um edge node acontece diretamente nos dispositivos quando o [Edge Orchestrator Agent](/pt-br/documentacao/produtos/guias/deploy/instalar-orchestrator-agent/) é instalado neles.
:::

## Passo 1: Crie um edge service

Para criar um edge service, você precisa dar a ele um nome fácil de lembrar.

<LinkButton link="/pt-br/documentacao/produtos/guias/deploy/criar-edge-service/" label="consulte o guia para criar um edge service" outlined />

---

## Passo 2: Crie um recurso

Para poder orquestrar serviços em seu dispositivo, você precisa configurar todos os recursos necessários para instalar, desinstalar e recarregar seus serviços.

Para criar um recurso, você precisa:

- Dar ao recurso um nome fácil de lembrar.
- Escolher o `content-type` do recurso. Exemplo: `text` e `shell-script`.
- Inserir o conteúdo do recurso.

<LinkButton link="/pt-br/documentacao/produtos/guias/deploy/provisionar-arquivos/" label="consulte o guia para criar um edge service resource" outlined />

---

## Passo 3: Gere uma credencial 

Para poder gerenciar edge nodes, você deve gerar as credenciais usadas pelo Edge Orchestrator Agent.

Para fazer isso: 

- Escolha uma descrição.
- Dê um nome fácil de lembrar.
- Defina a credencial como ativa.

<LinkButton link="/pt-br/documentacao/produtos/guias/deploy/gerar-credentials/" label="consulte o guia para criar uma credencial" outlined />
<br/>
<LinkButton link="/pt-br/documentacao/produtos/deploy/credenciais/" label="saiba mais sobre as credenciais" outlined />

---

## Passo 4: Instale o Agente do Edge Orchestrator

Com as credenciais criadas, baixe e instale o Agente do Edge Orchestrator.

<LinkButton link="/pt-br/documentacao/produtos/guias/deploy/instalar-orchestrator-agent/" label="consulte o guia de instalação do edge orchestrator agent" outlined />

## Passo 5: Autorize um edge node

Com o Agente do Edge Orchestrator instalado no dispositivo, autorize o edge node.

<LinkButton link="/pt-br/documentacao/produtos/guias/deploy/autorizar-edge-node/" label="consulte o guia para autorizar um edge node" outlined />

## Passo 6: Vincule um edge service a um edge node

Agora, com o edge node criado e autorizado, você deve vinculá-lo a um edge service a ser implantado no edge node.

<LinkButton link="/pt-br/documentacao/produtos/guias/deploy/vincular-servico/" label="consulte o guia para vincular um edge service a um edge node" outlined />

Uma vez que essa relação estiver estabelecida, quando uma atualização for acionada nos edge services, ela será implantada em todos os nodes relacionados a este serviço em particular.
