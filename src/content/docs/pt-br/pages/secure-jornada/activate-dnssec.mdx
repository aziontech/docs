**Domain Name System Security Extensions (DNSSEC)** fornecem uma camada extra de segurança para verificar a autenticidade e a integridade de uma resposta de endereço IP. A Azion fornece [compatibilidade com DNSSEC](/pt-br/documentacao/produtos/intelligent-dns/compatibilidade-dnssec/), desde que o registro do seu domínio de nível superior (TLD) suporte o recurso e [sua zona esteja configurada](/en/documentation/products/guides/secure/intelligent-dns-configure-main-settings/) com registros de recursos relacionados ao DNSSEC no [Intelligent DNS](/pt-br/documentacao/produtos/guias/secure/add-records/).

## Ativar DNSSEC via API

A ativação do DNSSEC está disponível via [API da Azion](https://api.azion.com/#f0f730a3-9739-40cf-ad07-0b7bca8abac8):

1. Execute a seguinte requisição `GET` no seu terminal, substituindo [TOKEN VALUE] pelo seu [personal token](/pt-br/documentacao/produtos/guias/personal-tokens/) para buscar seu `<hosted_zone_id>`:

```bash
curl --location 'https://api.azionapi.net/intelligent_dns' \
--header 'Authorization: Token [TOKEN VALUE]' \
--header 'Accept: application/json; version=3'
```

2. Você receberá uma resposta semelhante a esta:

```json
{
"count": 1,
"links": {
"previous": null,
"next": null
},
"total_pages": 1,
"results": [
{
"domain": "seudominio.com",
"is_active": true,
"name": "Uma zona hospedada",
"id": 1234
}
],
"schema_version": 3
}
```

3. Copie o valor `<id>` da zona específica na qual você deseja ativar o DNSSEC. Neste exemplo, é `1234`.
4. Execute a seguinte requisição `PATCH` no seu terminal, substituindo [TOKEN VALUE] pelo seu [personal token](/pt-br/documentacao/produtos/guias/personal-tokens/):

```bash
curl --location --request PATCH 'https://api.azionapi.net/intelligent_dns/hosted_zone_id/dnssec' \
--header 'Accept: application/json; version=3' \
--header 'Authorization: Token [TOKEN VALUE]' \
--header 'Content-Type: application/json' \
--data '{
"is_enabled": true
}'
```

5. Você receberá uma resposta semelhante a esta:

```json
{
"results": {
"is_enabled": true,
"status": "waiting"
},
"schema_version": 3
}
```

Agora, o DNSSEC está ativo na zona do Intelligent DNS que você especificou.

:::tip
Confira a [documentação da API da Azion](https://api.azion.com/) e a [especificação OpenAPI](https://github.com/aziontech/azionapi-openapi/) para saber mais sobre todos os recursos disponíveis via API.
:::
