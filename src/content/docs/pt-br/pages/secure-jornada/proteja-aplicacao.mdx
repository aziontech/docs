**Edge Firewall** permite que você proteja suas [edge applications](/pt-br/documentacao/produtos/edge-application/) e [domínios](/pt-br/documentacao/produtos/edge-application/domains/) contra vários tipos de ameaças e ataques.

A Azion fornece uma gama de ferramentas para configurar seu edge firewall:

- [Azion Console](https://console.azion.com/): fornece uma interface amigável para configurar seu edge firewall e conjuntos de regras.
- [Azion API](https://api.azion.com/#00268e07-463f-4de1-9b45-ceeab187bf19): permite criar, configurar e automatizar edge firewalls e seus módulos e recursos relacionados.
- [Azion SDK](/pt-br/documentacao/produtos/secure/automarizar/sdk/): permite melhorar o desenvolvimento de aplicações Go.
- [Terraform](/pt-br/documentacao/produtos/secure/automarizar/terraform/): fornece integração para sua infraestrutura com Azion usando Terraform. Configurações disponíveis para main settings e edge functions instances.
- [Edge Runtime](/pt-br/documentacao/produtos/secure/automarizar/functions/): aproveita APIs para manipulação de rede e dados.

---

## Passo 1: Defina as configurações principais

Para criar um edge firewall, o primeiro passo é fornecer:

- Um nome único e fácil de lembrar.
- Os domínios que você deseja associar ao edge firewall.
- Os módulos que você deseja usar.
    - Este é um passo opcional, mas é recomendado habilitar todos os módulos disponíveis para proteger totalmente suas aplicações.

Opcionalmente, você também pode habilitar o recurso **Debug Rules**.

## Passo 2: Escolha uma função para instanciar

O segundo passo é instanciar uma edge function.

Requer:

<Badge variant="accent">[Edge Functions no Edge Firewall](/pt-br/documentacao/produtos/edge-firewall/edge-functions/firewall/)</Badge>

Instanciar uma edge function criada para fins de segurança, como as integrações **Bot Manager** ou **Scheduled Blocking**, reforça suas políticas de segurança e adiciona outra camada de proteção para suas aplicações.

O passo de instanciação também permite que você aplique lógica específica e personalizada às suas edge functions.

<Button href="/pt-br/documentacao/produtos/guias/secure/instanciar-edge-functions/" text="saiba mais sobre instanciar edge functions" variant="secondary"></Button> <br></br>

## Passo 3: Configure variáveis no Rules Engine

O passo final é configurar variáveis no **Rules Engine**. Através de critérios (*criteria*) e comportamentos (*behaviors*), você configura as variáveis que devem ser executadas para proteger suas aplicações e a lógica na qual elas devem ser executadas.

As edge functions e as regras são executadas em paralelo. É recomendado usar ambas as ferramentas para melhorar ainda mais e garantir a segurança de sua aplicação.
