Neste guia, você aprenderá a criar uma zona [intelligent DNS](/pt-br/documentacao/produtos/intelligent-dns/) com suas principais configurações e editá-las.

---

## Via RTM

Nesta seção, você criará ou editará as configurações principais do **Intelligent DNS** com um domínio via [Real-Time Manager (RTM)](https://manager.azion.com).

1. [Acesse o RTM](/pt-br/documentacao/produtos/guias/como-acessar-o-rtm/) > **Intelligent DNS**.
2. Para criar uma nova zona, clique no botão **Add Zone**. Para editar uma zona existente, selecione o Intelligent DNS que deseja editar na lista.

:::caution[atenção]
Ao hospedar seu domínio no Intelligent DNS da Azion, você deve alterar os servidores DNS autoritativos do seu domínio para usar os nameservers da Azion:

- ns1.aziondns.net
- ns2.aziondns.com
- ns3.aziondns.org
:::

### Criando uma nova zona

1. Dê ao seu Intelligent DNS um nome único e fácil de lembrar.
2. Em **Domain**, adicione o nome do domínio que deseja hospedar no Intelligent DNS. Por exemplo: `meudominio`.
3. Clique no botão **Save**.

### Editando uma zona existente

1. Opcionalmente, edite o nome do seu Intelligent DNS.
2. Em **Domain Name**, edite o nome do domínio que deseja hospedar no Intelligent DNS.
3. Clique no botão **Save**.

Depois de configurar suas principais configurações, você pode adicionar registros ao seu Intelligent DNS.

## Via API

1. Execute a seguinte solicitação `POST` no seu terminal, substituindo `[VALOR DO TOKEN]` pelo seu [token pessoal](/pt-br/documentacao/produtos/guias/personal-tokens/) para criar um novo Intelligent DNS:

```bash
curl --location 'https://api.azionapi.net/intelligent_dns' \
--header 'Authorization: Token [VALOR DO TOKEN]' \
--header 'Accept: application/json; version=3' \
--header 'Content-Type: application/json' \
--data '{
"name": "Nova zona",
"domain": "seudominio.com",
"is_active": true
}'
```


2. Você receberá uma resposta semelhante a esta:

```json 
{
"results": [
{
"nx_ttl": 3600,
"domain": "seudominio.com",
"retry": 7200,
"name": "Nova zona",
"nameservers": [
"ns1.aziondns.net",
"ns2.aziondns.com",
"ns3.aziondns.org"
],
"soa_ttl": 3600,
"is_active": true,
"refresh": 43200,
"expiry": 1209600,
"id": 1234
}
],
"schema_version": 3
}
```


:::caution[atenção]
Uma vez que você hospede seu domínio no Intelligent DNS da Azion, você deve alterar os servidores DNS autoritativos do seu domínio para usar os nameservers da Azion:

- ns1.aziondns.net
- ns2.aziondns.com
- ns3.aziondns.org
:::

Aguarde alguns minutos para que as alterações se propaguem e seu Intelligent DNS será criado.

:::tip
Confira a [documentação da API da Azion](https://api.azion.com/) e a [especificação OpenAPI](https://github.com/aziontech/azionapi-openapi/) para saber mais.
::: 

Editando uma zona existente

1. Execute a seguinte solicitação PATCH no seu terminal, substituindo [VALOR DO TOKEN] pelo seu token pessoal e [ID DA ZONA] pelo ID da zona que deseja editar:

```
curl --location 'https://api.azionapi.net/intelligent_dns/[ID DA ZONA]' \
--request PATCH \
--header 'Authorization: Token [VALOR DO TOKEN]' \
--header 'Accept: application/json; version=3' \
--header 'Content-Type: application/json' \
--data '{
    "name": "Zona editada",
    "domain": "seudominioeditado.com",
    "is_active": true
}'
``` 

2. Você receberá uma resposta semelhante a esta:

```
{
  "results": [
    {
      "nx_ttl": 3600,
      "domain": "seudominioeditado.com",
      "retry": 7200,
      "name": "Zona editada",
      "nameservers": [
        "ns1.aziondns.net",
        "ns2.aziondns.com",
        "ns3.aziondns.org"
      ],
      "soa_ttl": 3600,
      "is_active": true,
      "refresh": 43200,
      "expiry": 1209600,
      "id": 1234
    }
  ],
  "schema_version": 3
}
``` 

Aguarde alguns minutos para que as alterações se propaguem e seu Intelligent DNS será atualizado.

:::tip
Confira a [documentação da API da Azion](https://api.azion.com/) e a [especificação OpenAPI](https://github.com/aziontech/azionapi-openapi/) para saber mais.
::: 

Depois de configurar suas principais configurações, você pode adicionar registros ao seu Intelligent DNS.
