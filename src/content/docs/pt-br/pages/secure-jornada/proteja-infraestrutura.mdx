Após [criar um edge firewall e proteger sua aplicação](/pt-br/documentacao/produtos/guias/secure/edge-firewall-definir-main-settings/), você pode usar o **Origin Shield** para proteger sua infraestrutura bloqueando ameaças.

O **Origin Shield** funciona como uma lista de endereços IP/CIDR usados no edge da Azion, fornecendo, assim, a garantia de IPs seguros para permitir o acesso. A lista é mantida pela Azion e está disponível via Azion Console e API, mas cada cliente deve criar automação para garantir que suas políticas de segurança estejam sempre atualizadas com os endereços do Origin Shield.

---

## Passo 1. Ative o Origin Shield

Para acessar a lista do Origin Shield, você deve ter:

- Um [edge firewall criado](/pt-br/documentacao/produtos/guias/secure/edge-firewall-definir-main-settings/).
- O [módulo Network Layer Protection](/pt-br/documentacao/produtos/guias/secure/edge-firewall-definir-main-settings/#modulos) ativado.

---

## Passo 2. Acesse a lista de endereços

### Via Azion Console

1. [Acesse o Azion Console](/pt-br/documentacao/produtos/guias/como-acessar-o-rtm/) > selecione **Network Lists**.
2. Selecione **Azion IP Tor Exit Nodes**.
3. No campo **List**, você pode copiar a lista de endereços IP.

:::note
Clientes com Origin Shield recebem um email toda vez que a lista é atualizada.
:::

### Via API

1. Execute a seguinte requisição `GET` em seu terminal, substituindo `[TOKEN VALUE]` pelo seu [personal token](/pt-br/documentacao/produtos/guias/personal-tokens/):

```bash
curl --location 'https://api.azionapi.net/network_lists' \
--header 'Accept: application/json; version=3' \
--header 'Authorization: Token [TOKEN VALUE]'
```

2. Você receberá uma resposta semelhante a esta:

```json
{
  "count": 4,
  "total_pages": 1,
  "schema_version": 3,
  "links": {
    "previous": null,
    "next": null
  },
  "results": [
    {
      "id": 2345,
      "last_editor": "last.editor@azion.com",
      "last_modified": "2023-03-27T21:19:27.996870Z",
      "list_type": "ip_cidr",
      "name": "Azion Origin Shield",
      "country_list": [],
      "ip_list": [
        "192.168.0.5"
      ]
    }
  ]
}
```

3. No campo `ip_list`, você receberá todos os IPs adicionados pela Azion. Você pode copiar a lista de endereços IP.

---
