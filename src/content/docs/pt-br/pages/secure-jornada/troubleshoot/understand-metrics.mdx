Uma vez que você cria um edge firewall e [ativa o módulo Web Application Firewall (WAF)](/pt-br/documentacao/produtos/guias/secure/definir-configuracoes-principais/#modulos), você também pode usar os produtos de [Observe](/pt-br/documentacao/produtos/observe/) para monitorar como o WAF está processando as requisições feitas às suas aplicações.

- [Real-Time Metrics](/pt-br/documentacao/produtos/real-time-metrics/): acesso a gráficos com dados agregados para explorar comportamentos durante períodos mais longos de armazenamento de dados.
- [GraphQL API](/pt-br/documentacao/produtos/graphql-api/visao-geral/): acesso a dados agregados e brutos via API com consultas personalizadas com os dados solicitados.
- [Real-Time Events](/pt-br/documentacao/produtos/real-time-events/): acesso a logs com dados brutos para realizar análises extensivas.
