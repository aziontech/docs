Uma vez que você hospeda seus domínios e cria zonas no [Intelligent DNS](/pt-br/documentacao/produtos/intelligent-dns/), você também pode contar com os produtos de [Observe](/pt-br/documentacao/produtos/observe/) para monitorar como o WAF está processando as solicitações feitas às suas aplicações.

- [Real-Time Metrics](/pt-br/documentacao/produtos/real-time-metrics/): acesso a gráficos com dados agregados para explorar comportamentos ao longo de períodos mais longos de armazenamento de dados.
- [GraphQL API](/pt-br/documentacao/produtos/graphql-api-overview/): acesso a dados agregados e brutos via API com consultas personalizadas com apenas os dados solicitados.
