Depois de criar uma edge application na Azion, é possível editar e gerenciar suas configurações sempre que quiser, incluindo a exclusão dela.

Para excluir uma edge application existente, siga estes passos:

1. Acesse o [Azion Console](/pt-br/documentacao/produtos/guias/como-acessar-o-azion-console/) > **Edge Application**.
    - Você será redirecionado para a página de **Edge Application** que lista todas as edge applications que você criou.
    - A lista é organizada em ordem alfabética. Você também pode usar a **barra de busca** localizada no canto superior esquerdo da lista; atualmente, ela é filtrada apenas pelo **Name** (nomes das edge applications).
2. Encontre a edge application que você deseja excluir.
3. Clique nos três pontos horizontais no canto direito.
4. Selecione a opção **Delete**.
5. Digite "delete" no popover para confirmar que deseja excluir essa aplicação específica.
6. Clique no botão **Delete**.

Pronto. Uma mensagem de sucesso aparecerá na parte superior da página confirmando que sua edge application foi excluída e não aparecerá mais na sua lista.
