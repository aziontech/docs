Depois de criar uma application na Azion, é possível editar e gerenciar suas configurações sempre que quiser, incluindo fazer a clonagem dela. Nesse caso, a nova application terá as mesmas configurações da primeira.

Para clonar uma application existente, siga estes passos:

1. Acesse o [Azion Console](/pt-br/documentacao/produtos/guias/como-acessar-o-azion-console/) > **Applications**.
    - Você será redirecionado para a página de **Applications** que lista todas as applications que você criou.
    - A lista é organizada em ordem alfabética. Você também pode usar a **barra de busca** localizada no canto superior esquerdo da lista.
2. Encontre a application que você deseja clonar.
3. Clique nos três pontos horizontais no canto direito.
4. Selecione a opção **Clone**.
5. Digite um nome para sua nova edge aplication no popover.
    - Digite um nome único e fácil de lembrar.
    - A plataforma retorna uma mensagem de erro se o nome já estiver sendo usado. Se isso acontecer, tente novamente com um nome diferente.
6. Clique no botão **Clone**.
7. Você será redirecionado para a página **Editar Application**. Clique no botão **Save** para salvar sua application.

Pronto. Uma mensagem de sucesso aparecerá na parte superior da página confirmando que sua application foi clonada.

A application clonada aparecerá na lista e terá as mesmas configurações que a original.
