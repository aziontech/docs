Depois de criar uma edge application na Azion, é possível editar e gerenciar suas configurações sempre que quiser, incluindo fazer a clonagem dela. Nesse caso, a nova edge application terá as mesmas configurações da primeira.

Para clonar uma edge application existente, siga estes passos:

1. Acesse o [Azion Console](/pt-br/documentacao/produtos/guias/como-acessar-o-azion-console/) > **Edge Application**.
    - Você será redirecionado para a página de **Edge Application** que lista todas as edge applications que você criou.
    - A lista é organizada em ordem alfabética. Você também pode usar a **barra de busca** localizada no canto superior esquerdo da lista.
2. Encontre a edge application que você deseja clonar.
3. Clique nos três pontos horizontais no canto direito.
4. Selecione a opção **Clone**.
5. Digite um nome para sua nova edge aplication no popover.
    - Digite um nome único e fácil de lembrar.
    - A plataforma retorna uma mensagem de erro se o nome já estiver sendo usado. Se isso acontecer, tente novamente com um nome diferente.
6. Clique no botão **Clone**.
7. Você será redirecionado para a página **Editar Edge Application**. Clique no botão **Save** para salvar sua edge application.

Pronto. Uma mensagem de sucesso aparecerá na parte superior da página confirmando que sua edge application foi clonada.

A edge application clonada aparecerá na lista e terá as mesmas configurações que a original.
