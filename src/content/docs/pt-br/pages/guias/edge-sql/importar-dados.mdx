Este guia explica como importar dados para o seu banco de dados usando o Edge SQL Shell.

## Requisitos

- **Edge SQL Shell instalado**: Certifique-se de que o Edge SQL Shell está devidamente [instalado em seu sistema](/pt-br/documentacao/produtos/store/sql/install-edge-sql-shell/).

- **Banco de dados**: Você precisa ter um banco de dados criado. Para criar um novo banco de dados no Edge SQL usando o Shell, use o comando:
  
```sh
  .create <database-name>
```

<slot name="file" />

1. Selecione um banco de dados.

```shell
.use database-name
```

2. Importe seu arquivo.

```shell
.import file <csv|xlsx> <file_path> <table_name>
```
Para verificar a criação de sua tabela você pode usar:

```shell
.tables
```

<fragment slot="panel.kaggle">
<slot name="Kaggle" />

Use os seguintes comandos para definir variáveis de ambiente com suas credenciais do Kaggle:

```sh
  export KAGGLE_USERNAME="<username>"
  export KAGGLE_KEY="<kaggle_api_key>"
```

1. Selecione um banco de dados.

```shell
.use database-name
```

1. Importe seu banco de dados kaggle.

```shell
.import kaggle <dataset> <data_name> <table_name>
```
Para verificar a criação de sua tabela você pode usar:

```shell
.tables
```
</fragment>

<fragment slot="panel.mysql">
<slot name="MySQL" />

Use os seguintes comandos para definir variáveis de ambiente com suas credenciais do MySQL:

```sh
  export MYSQL_USERNAME="<username>"
  export MYSQL_PASSWORD="<password>"
  export MYSQL_HOST="<host_address>"
```

Configurações opcionais para MySQL:

```sh
  export MYSQL_PORT=<port>
  
  # For TLS connection
  export MYSQL_SSL_CA="<ssl_ca>"
  export MYSQL_SSL_CERT="<ssl_cert>"
  export MYSQL_SSL_KEY="<ssl_key>"
  export MYSQL_SSL_VERIFY_CERT=<True|False>
```

1. Selecione um banco de dados.

```shell
.use database-name
```

2. Importe seu banco de dados MySQL.

```shell
.import mysql <database> <source_table> <table_name>
```
Para verificar a criação de sua tabela você pode usar:

```shell
.tables
```
</fragment>

## Importando dados a partir de um arquivo SQL

Para importar dados de um arquivo `.sql` contendo comandos SQL `INSERT`, use o comando `.read` no Edge SQL Shell. Isso executará as instruções SQL contidas no arquivo e preencherá seu banco de dados com os dados especificados.

```shell
.read <file_name>
```
