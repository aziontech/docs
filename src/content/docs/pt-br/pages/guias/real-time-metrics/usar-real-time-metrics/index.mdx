O **Real-Time Metrics** fornece acesso, em tempo real, a métricas por meio de gráficos. Os gráficos exibem dados assim que suas aplicações e outros produtos começam a ter acessos e tráfego de entrada.

:::note[nota]
O novo Real-Time Metrics fornece dados e métricas a partir de **15 de outubro de 2022**. Se você quiser visualizar métricas de até 2 anos e antes de 15 de outubro de 2022, consulte o [Real-Time Metrics Histórico](/pt-br/documentacao/produtos/real-time-metrics-historico/).
:::

---

## Configurar produtos e intervalo de data

Acesse o [Real-Time Manager (RTM)](https://manager.azion.com/) e selecione **Products menu** > **Real-Time Metrics** na seção **OBSERVE**.

Para analisar suas métricas, primeiro, você precisa selecionar um produto e configurar um intervalo de tempo:

1. Selecione uma das três categorias disponíveis no menu suspenso:

   <Badge>Build</Badge> <Badge>Secure</Badge> <Badge>Observe</Badge>

2. Selecione uma aba de acordo com o produto que deseja visualizar:

- **Build**

   <Badge>Edge Applications</Badge> <Badge>L2 Caching</Badge> <Badge>Edge Functions</Badge> <Badge>Image Processor</Badge>

- **Secure**

   <Badge>WAF</Badge> <Badge>Intelligent DNS</Badge>

- **Observe**

   <Badge>Data Streaming</Badge>

3. Em **Time range**, selecione um período de tempo no menu suspenso para buscar os dados que serão exibidos nos gráficos:

   <Badge>Last Hour (última hora)</Badge> <Badge>Last 24 Hours (últimas 24 horas)</Badge> <Badge>Last 7 Days (últimos 7 dias)</Badge> <Badge>Last 30 Days (últimos 30 dias)</Badge> <Badge>Last 6 Months (últimos 6 meses)</Badge>

4. Se você quiser usar uma data diferente das opções, clique nos campos do calendário e selecione uma data e hora de início e término.
5. Se você selecionou a aba do produto **Edge Applications**, selecione entre uma das quatro subabas:

   <Badge>Data Transferred</Badge> <Badge>Requests</Badge> <Badge>Status Codes</Badge> <Badge>Bandwidth Saving</Badge>

Todos os gráficos de todas as abas são atualizados automaticamente após a aplicação de um intervalo de tempo.

## Visualizar gráficos e métricas

Depois de selecionar uma aba para visualizar gráficos, você pode:

- Adicionar filtros.
- Escolher um intervalo de dados.
- Usar as tags **Get help** para saber mais sobre cada gráfico.

Para analisar um gráfico:

1. Navegue pela página para ver todos os gráficos disponíveis.
2. Passe o cursor sobre os pontos dos gráficos para visualizar os dados e o intervalo de tempo específicos.
3. Aumente e diminua o zoom no gráfico rolando para cima e para baixo.
4. Use o **Context menu** para copiar a query do gráfico, exportar os pontos em um arquivo `.csv`, habilitar a linha média do gráfico ou habilitar as linhas médias do gráfico por série.
5. Analise a **tag de Variation** para ver como seus dados estão se comportando em relação ao intervalo de dados correspondente anterior.
6. Copie o caminho da URL para compartilhar os filtros que você aplicou com outros usuários do RTM.

Se você deseja complementar sua análise com informações mais detalhadas sobre as métricas visualizadas nos gráficos, veja os registros do Real-Time Events.
