Durante a [Transição para a API v4](/pt-br/documentacao/produtos/api-v4-migration/), você pode ver algumas páginas com versões diferentes para v3 e v4. Este guia ajuda você a verificar se sua conta já foi migrada para a v4.

1. Acesse o [Azion Console](/pt-br/documentacao/produtos/guias/como-acessar-o-azion-console/).
2. Abra o **Products menu**.

Se você visualizar opções como **Workloads**, **Connectors** ou **Custom Pages**, sua conta foi migrada para a v4 e você pode usar os novos produtos.
