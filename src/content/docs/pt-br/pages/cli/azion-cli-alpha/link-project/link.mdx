Veja o comando que permite a vinculação de um projeto existente para ser executado na plataforma de Edge da Azion.

---

## link

O processo de vinculação adapta o ambiente para a construção de sua edge application.

### Uso

```sh
  azion link
```

### Saída 

Depois de inicializar a vinculação de uma aplicação existente, a lista de modos disponíveis será exibida. Em seguida, é necessário escolher um deles.

```sh 
  Escolha um modo:  [Use as setas para mover, digite para filtrar]
  > Html (Deliver)
    Javascript (Compute)
    Typescript (Compute)
    Angular (Deliver)
    Astro (Deliver)
    Hexo (Deliver)
    Next (Deliver)
```

### Flags opcionais

#### name

A opção `--name` pode ser especificada, mas não é obrigatória. Se não for informado, o nome do seu projeto será gerado automaticamente.

#### mode

A opção `--mode` pode ser especificada, mas não é obrigatória. Os valores esperados são `deliver` e `compute`.

#### preset

A opção `--preset` pode ser especificada, mas não é obrigatória. Os valores esperados são:

- Html
- Javascript
- Typescript
- Angular
- Astro
- Hexo
- Next 
- React
- Vue
- static

#### help

A opção `--help` exibe mais informações sobre o comando `link`.
