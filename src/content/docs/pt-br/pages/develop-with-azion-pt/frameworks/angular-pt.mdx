Angular é um framework web de código aberto projetado com ênfase em velocidade e adequação para sites que contenham conteúdo substancial, como:

- Páginas de destino
- Blogs
- Documentação técnica
- Plataformas online similares

Saiba mais sobre o [Angular](https://angular.io/docs).

## Requisitos

Antes de começar, você deve ter:

- Uma conta na [plataforma Azion](/pt/documentacao/produtos/contas/criacao-de-conta/) com [Funções de Borda (Edge Functions)](/pt/documentacao/produtos/aplicacao-de-borda/funcoes-de-borda/) habilitadas.
- [A versão mais recente do Azion CLI instalada](/pt/documentacao/produtos/azion-cli/visao-geral/).
- Um editor de código.
- Acesso ao terminal.

## Inicializando um projeto Angular na borda

1. Inicialize o projeto:

```sh 
  azion init
```

2. Aceite o nome sugerido para o seu projeto ou digite um nome:

```sh 
(Pressione Enter para aceitar o nome sugerido entre parênteses) Nome do seu projeto:  (dynamic_caligari)
```

Saída:

```sh
Obtendo modos disponíveis
```

3. Escolha o modelo Angular:

```sh 
Obtendo modos disponíveis? Escolha um modelo para o seu projeto: (Use as setas)
 ❯Angular 
  Astro 
  Hexo 
  Next 
  React 
  Vue 
  Vite 
```

Agora, o CLI do Angular seguirá o processo e inicializará seu projeto Angular.

4. Após a inicialização ser concluída, escolha o modo:

```sh 
Escolha um modo:  [Use as setas para mover, digite para filtrar]
  Javascript (Computação)
  Typescript (Computação)
 ❯Angular (Entrega)
  Astro (Entrega)
  Hexo (Entrega)
  Next (Entrega)
  React (Entrega)
```

5. Com o modelo baixado e configurado, você pode iniciar um servidor de desenvolvimento local agora.

```sh 
Você deseja iniciar um servidor de desenvolvimento local? (s/N) 
```

As etapas a seguir são baseadas na resposta que você forneceu.

- [Se você escolheu ter um servidor de desenvolvimento local em execução]().
- [Se você escolheu não ter um servidor de desenvolvimento local em execução]().

### Responder sim para desenvolvimento local

1. Instale as dependências do projeto. Digite `s` quando a interação pedir:

```sh 
  Você deseja instalar as dependências do projeto? Isso pode ser necessário para iniciar o servidor de desenvolvimento local (s/N)
```

Aguarde até que a instalação seja concluída.

Saída:

```sh
Sua Aplicação de Borda foi construída com sucesso
[Vulcan] [Servidor] › ✔  sucesso   Função em execução na porta http://localhost:3000/
```

2. No navegador, vá para `http://localhost:3000/` e você poderá ver seu projeto Angular em execução.

#### Implantação do projeto

Quando seu projeto estiver em execução localmente, você ainda poderá implantá-lo na borda. Para fazer isso:

1. Pare a execução do terminal com `Control + C`. 
2. Acesse a pasta do projeto:

```sh
cd [nome-do-seu-projeto] 
```

3. Implante o projeto:

```sh
azion deploy
```

4. Aguarde enquanto o projeto é construído e implantado na Plataforma Azion Edge.

Após a conclusão da implantação, você receberá um domínio para acessar seu projeto Angular na plataforma da Azion.

Aguarde alguns minutos para que a propagação ocorra e, em seguida, acesse sua aplicação usando o domínio fornecido, que deve ser semelhante a `https://xxxxxxx.map.azionedge.net`.

### Responder não para desenvolvimento local

Após indicar que você não deseja que um servidor local seja executado, implante o projeto Angular na borda.

1. Digite `s` na seguinte interação, indicando que deseja implantar o projeto:

```sh
   Você deseja implantar o projeto? (s/N) 
```

2. Instale as dependências do projeto. Digite `s` quando a interação pedir:

```sh 
  Você deseja instalar as dependências do projeto? Isso pode ser necessário para implantar o projeto (s/N)
```

3. Aguarde enquanto o projeto é construído e implantado na Plataforma Azion Edge.

Após a conclusão da implantação, você receberá um domínio para acessar seu projeto Angular na plataforma da Azion.

Aguarde alguns minutos para que a propagação ocorra e, em seguida, acesse sua aplicação usando o domínio fornecido, que deve ser semelhante a `https://xxxxxxx.map.azionedge.net`.
