:::caution[importante]
A Azion possui duas interfaces de usuário: Real-Time Manager e Console. Siga os passos de acordo com a interface de usuário que estiver utilizando.
:::
:::caution[importante]
**Atenção**: O Real-Time Manager será **descontinuado** a partir de **05 de dezembro de 2024**. Todos os usuários terão acesso apenas ao Console a partir dessa data.
:::
